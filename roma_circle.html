Уравнение окружности

Стандартный вид: (x − a)² + (y − b)² = R²

где (a; b) — координаты центра, R — радиус окружности.

📏

Расстояние между центрами

Формула расстояния:
d = √((x₂ − x₁)² + (y₂ − y₁)²)

Вычисляется по координатам центров O₁ и O₂.

🔀

Три случая расположения

1. Нет общих точек
2. Одна общая точка (касание)
3. Две общие точки (пересечение)

Условия взаимного расположения

Случай Условие Описание
Нет общих точек d > r₁ + r₂ Окружности вне друг друга
Нет общих точек d < |r₁ − r₂| Одна внутри другой
Касание (1 точка) d = r₁ + r₂ Внешнее касание
Касание (1 точка) d = |r₁ − r₂| Внутреннее касание
Пересечение (2 точки) |r₁ − r₂| < d < r₁ + r₂ Две общие точки
Как привести уравнение к стандартному виду? +

Для приведения общего вида уравнения x² + y² + Dx + Ey + F = 0 к стандартному:


1. Сгруппируйте слагаемые с x и с y отдельно

2. Выделите полный квадрат для каждой группы

3. Перенесите свободный член в правую часть


Пример: x² − 2x + y² − y − 1 = 0

(x² − 2x + 1) + (y² − y + 0.25) = 1 + 1 + 0.25

(x − 1)² + (y − 0.5)² = 2.25


Центр: (1; 0.5), Радиус: √2.25 = 1.5

Уравнения с произведением переменных +

Уравнение вида x² + Bxy + Cy² + Dx + Ey + F = 0 может раскладываться на множители.


Алгоритм:

1. Рассмотрите как квадратное относительно x

2. Вычислите дискриминант D

3. Если D — полный квадрат, уравнение раскладывается


Пример: x² − xy − 6y² = 0

D = y² + 24y² = 25y² = (5y)²

Корни: x₁ = 3y, x₂ = −2y

График — две прямые: y = x/3 и y = −x/2

Графический метод решения неравенств +

Алгоритм решения:


1. Постройте граничные линии (замените знак на =)

2. При строгом неравенстве (<, >) линии пунктирные

3. При нестрогом (≤, ≥) линии сплошные

4. Граничные линии разбивают плоскость на области

5. Методом подстановки проверьте каждую область

6. Закрасьте подходящие области


Важно: Для проверки берите простые точки с нулевыми координатами.

Задача 1

Даны окружности:
ω₁: x² − 2x + y² − y − 1 = 0
ω₂: (x − 3)² + (y + 5)² = 16

Решение:
O₁(1; 0.5), r₁ = 1.5
O₂(3; −5), r₂ = 4
d = √34.25 ≈ 5.85
d > r₁ + r₂ → нет общих точек

📝

Задача 2

Разложите на множители:
x² − xy − 6y² = 0

Решение:
D = y² + 24y² = 25y²
x₁ = 3y, x₂ = −2y
Две прямые: y = x/3 и y = −x/2

📝

Задача 3

Решите неравенство:
x² − xy − 6y² > 0

Решение:
Границы: y = x/3, y = −x/2
4 области, проверка подстановкой
Подходят 2 противоположные области
