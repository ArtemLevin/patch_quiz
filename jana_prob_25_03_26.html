Попадание случайной величины в интервал

Интерактивное пособие по вычислению вероятностей попадания непрерывной случайной величины в заданный промежуток

P(a ≤ X < b) = P(X < b) − P(X < a)

Теоретический материал

Основная формула вероятности интервала

Для нахождения вероятности попадания непрерывной случайной величины X в полуинтервал [a; b) используется формула:

P(a ≤ X < b) = P(X < b) − P(X < a)

Эта формула следует из свойств вероятности и аддитивности несовместных событий.

Свойство противоположных событий

Если известна вероятность P(X ≥ a), то вероятность противоположного события вычисляется как:

P(X < a) = 1 − P(X ≥ a)

Это свойство часто необходимо, когда в условии задачи дана вероятность одного события, а для решения требуется вероятность противоположного.

Геометрический смысл

Вероятность попадания в интервал можно интерпретировать как площадь под кривой плотности распределения на этом промежутке. Чем больше площадь, тем выше вероятность.

Сумма вероятностей всех непересекающихся областей равна 1 (или 100%), что соответствует полной вероятности.

3
Примените основную формулу

Вычтите из P(X < b) вероятность P(X < a) для получения P(a ≤ X < b).

4
Проверьте результат

Убедитесь, что полученная вероятность лежит в пределах от 0 до 1.

Шпаргалка

Основные формулы

P(a ≤ X < b) = P(X < b) − P(X < a)
P(X < a) = 1 − P(X ≥ a)
P(X ≥ a) = 1 − P(X < a)
P(X ≤ b) = P(X < b) (для непрерывных)

Важные свойства

  • ✓ 0 ≤ P(A) ≤ 1
  • ✓ P(достоверное) = 1
  • ✓ P(невозможное) = 0
  • ✓ P(A) + P(не A) = 1
  • ✓ Сумма вероятностей всех исходов = 1
