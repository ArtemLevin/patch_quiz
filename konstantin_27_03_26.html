Теория в аккордеонах

Ниже собран компактный теоретический блок: метод узловых точек, работа с масштабом, поиск точки пересечения, замена переменной и алгоритм распознавания квадратного корня.

Узловая точка — это точка графика, координаты которой удобно считать. Если функция известна с точностью до коэффициентов, то эти координаты подставляются в общую формулу, и дальше решается уравнение или система.

  • Один неизвестный коэффициент → достаточно одной точки.
  • Два неизвестных коэффициента → нужны две точки.
  • Перед вычислениями нужно убедиться, что точка действительно удобна.
Пример: y = a√x, точка (4; 3). Тогда 3 = a√4 = 2a, значит a = 3 / 2.

Одна клетка не всегда равна единице. На графиках с тригонометрией особенно важно смотреть, сколько клеток соответствует π, π / 2 или π / 4.

  • Если 4 клетки соответствуют π, то 1 клетка = π / 4.
  • Если точка отстоит на 2 клетки от нуля, то x = 2·(π / 4) = π / 2.
  • Масштаб по оси x и по оси y может быть разным.

После восстановления формул двух графиков нужно приравнять функции: f(x) = g(x). Решение этого уравнения даёт абсциссу точки пересечения.

  • Сначала ищем x.
  • Потом обязательно подставляем x в любую из формул.
  • Только после этого получаем ординату y.
Если спрашивают ординату точки пересечения, нельзя останавливаться на найденном x.

Когда в уравнении встречается √x и x, удобно ввести замену t = √x. Тогда x выражается как t², а не наоборот.

t = √x ⇒ x = t²

После замены получается квадратное уравнение относительно t. После его решения нужно помнить, что t = √x ≥ 0, поэтому отрицательные значения t сразу исключаются.

В занятии отдельно разбирался быстрый приём распознавания квадратного корня из натурального четырёхзначного числа.

  • Сначала ищем, между какими квадратами лежит число.
  • Потом смотрим на последнюю цифру числа.
  • Подбираем возможную последнюю цифру корня.
  • Подтверждаем обычной проверкой умножением.
Пример: 784 лежит между 20² = 400 и 30² = 900, последняя цифра 4, проверка даёт 28² = 784.
  • Если прямая убывает, её коэффициент k должен быть отрицательным.
  • Если график показывает положительную точку пересечения, а вычисления дают отрицательную, надо искать ошибку.
  • Если точка заявлена как узловая, но даёт слишком громоздкие числа, возможно, выбрана неудачная точка.
  • Если найденный корень после замены не удовлетворяет t = √x ≥ 0, его нельзя брать.

Разборы задач

Два типовых разбора по теме: сначала восстановление корневой функции и прямой, затем задача на тригонометрический график с учётом масштаба.

Разбор 1. Корневая функция и прямая

Пусть корневая функция проходит через точку (4; 3), а прямая — через точки (0; −4) и (8; 1). Найдём формулы графиков.

  1. Для корневой функции берём общий вид: y = a√x.
  2. Подставляем точку (4; 3): 3 = a√4 = 2a, значит a = 3 / 2.
  3. Получаем корневую функцию: y = (3 / 2)√x.
  4. Для прямой берём общий вид: y = kx + b.
  5. Из точки (0; −4) сразу получаем b = −4.
  6. Подставляем точку (8; 1): 1 = 8k − 4, значит 8k = 5, то есть k = 5 / 8.
Ответ: y = (3 / 2)√x и y = (5 / 8)x − 4.

Разбор 2. Тригонометрическая функция y = a·cos x + b

Пусть график проходит через точки (0; 1/2) и (π/2; −1). Найдём коэффициенты a и b.

  1. Записываем общий вид: y = a·cos x + b.
  2. Подставляем точку (0; 1/2): 1/2 = a·cos 0 + b = a + b.
  3. Подставляем точку (π/2; −1): −1 = a·cos(π/2) + b = b.
  4. Получаем b = −1.
  5. Тогда из первого уравнения: a − 1 = 1/2, значит a = 3/2.
Ответ: a = 3 / 2, b = −1, поэтому y = (3 / 2)·cos x − 1.

Cheat sheet

Сводка, которую удобно прокрутить перед самостоятельным решением.

Количество точек

Один неизвестный коэффициент → одна точка. Два неизвестных коэффициента → две точки.

y = a√x → достаточно одной узловой точки

Линейная функция

Сначала считываем b по оси Oy, затем находим k по второй точке.

y = kx + b

Замена переменной

В задачах с √x удобнее перейти к квадратному уравнению по новой переменной.

t = √x, x = t²

Ордината пересечения

После найденного x нужно обязательно подставить его в формулу.

Сначала x, потом y

Тригонометрия

Если по оси x клетками отложено π, сначала переводим клетки в радианы.

4 клетки = π → 1 клетка = π / 4

Табличные значения

Без них решения становятся медленными и уязвимыми для ошибок.

tg 0 = 0, tg(π / 4) = 1, cos 0 = 1, cos(π / 2) = 0

Корень из 4-значного числа

Оцени диапазон, проверь последнюю цифру, потом быстро подтверди умножением.

Например: √1296 = 36

Контроль знака

Убывающая прямая не может иметь положительный коэффициент наклона.

Убывает → k < 0
