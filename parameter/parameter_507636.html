Параметр в системе: когда решений ровно два?

Разберём задачу не только алгебраически, но и геометрически: первое уравнение задаёт две прямые, а второе — окружность с центром в точке (a, a). Нужно понять, при каких значениях параметра a суммарно получается ровно два пересечения.

{ 2x² + 2y² = 5xy,
(x − a)² + (y − a)² = 5a⁴. }
Ключевая идея: разложить первое уравнение на множители Дальше: расстояние от центра до каждой прямой Итог: один точный набор значений параметра

Пошаговое решение

Шаг 1. Преобразуем первое уравнение.

Перенесём всё в одну сторону:

2x² − 5xy + 2y² = 0.

Это однородное квадратное уравнение. Разложим на множители:

2x² − 5xy + 2y² = (2x − y)(x − 2y).

Значит, первое уравнение эквивалентно условию

(2x − y)(x − 2y) = 0,

то есть точка (x, y) должна лежать хотя бы на одной из двух прямых:

y = 2x   или   x = 2y (то есть y = x/2).
Геометрический смысл.
Всё множество решений первого уравнения — это не кривая второго порядка целиком, а две пересекающиеся прямые. Они проходят через начало координат и пересекаются в точке (0,0).
Шаг 2. Разберём второе уравнение.
(x − a)² + (y − a)² = 5a⁴.

Это окружность с центром

C(a, a)

и радиусом

R = √(5a⁴) = √5 · a².

Радиус всегда неотрицателен и зависит от a². При a = 0 окружность вырождается в одну точку (0,0).

Шаг 3. Сколько пересечений даёт окружность с каждой прямой?

Число точек пересечения окружности с прямой зависит от расстояния от центра окружности до этой прямой:

если d < R, то 2 точки;
если d = R, то 1 точка (касание);
если d > R, то 0 точек.
Шаг 4. Найдём расстояния от центра C(a, a) до обеих прямых.

Первая прямая: y = 2x, или 2x − y = 0.

Расстояние от точки (x₀, y₀) до прямой Ax + By + C = 0 равно

d = |Ax₀ + By₀ + C| / √(A² + B²).

Тогда для точки (a, a) получаем:

d₁ = |2a − a| / √(2² + (−1)²) = |a| / √5.

Вторая прямая: x = 2y, или x − 2y = 0.

d₂ = |a − 2a| / √(1² + (−2)²) = |−a| / √5 = |a| / √5.
Очень важный вывод: до обеих прямых центр окружности находится на одинаковом расстоянии. Значит, окружность ведёт себя по отношению к этим прямым симметрично.
Шаг 5. Сравниваем расстояние с радиусом.

Нужно сравнить

d = |a|/√5   и   R = √5 · a².

Условие касания к любой из прямых:

|a|/√5 = √5 · a².

Умножим на √5:

|a| = 5a².

Теперь разберём случаи.

Случай A: a = 0

Тогда окружность — это единственная точка (0,0).

Эта точка лежит на обеих прямых.

Значит, система имеет ровно одно решение.

Случай B: a ≠ 0

Можно разделить обе части уравнения |a| = 5a² на |a|:

1 = 5|a|,

откуда

|a| = 1/5,   то есть   a = ±1/5.
Шаг 6. Почему именно при a = ±1/5 решений ровно два?

При a = ±1/5 имеем d = R. Значит, окружность касается каждой из двух прямых.

Это даёт по одной точке касания на каждой прямой. Осталось убедиться, что это две разные точки, а не одна и та же.

Общая точка двух прямых только одна: (0,0). Но окружность при a = ±1/5 не проходит через начало координат, потому что

(0 − a)² + (0 − a)² = 2a²,

а по уравнению окружности должно быть

2a² = 5a⁴.

Для a = ±1/5 это неверно: левая часть равна 2/25, правая — 1/125.

Значит, точки касания разные, и всего решений действительно ровно два.

Шаг 7. Что происходит при остальных a?
УсловиеНа каждой прямойВсего решений
|a| = 1/5касание, 1 точка2
0 < |a| < 1/5d > R, пересечений нет0
|a| > 1/5d < R, по 2 точки4
a = 0вырожденная окружность в (0,0)1

Ответ

a = −1/5 и a = 1/5.

Только при этих двух значениях окружность касается обеих прямых, не проходя через их общую точку, поэтому система имеет ровно два решения.

Проверка через подстановку в прямые

Это полезная алгебраическая сверка.

Если y = 2x

Подставляем во второе уравнение:

(x − a)² + (2x − a)² = 5a⁴.

После раскрытия:

5x² − 6ax + 2a² − 5a⁴ = 0.

Дискриминант этого квадратного уравнения по x:

D = (−6a)² − 4·5·(2a² − 5a⁴) = 36a² − 40a² + 100a⁴ = 4a²(25a² − 1).

Для касания нужно D = 0, откуда снова получаем:

a = 0  или  |a| = 1/5.

Но a = 0 даёт одну общую точку обеих прямых, а a = ±1/5 даёт по одной точке на каждой прямой, то есть 2 решения всего.

Почему при |a| > 1/5 решений именно четыре?

Тогда дискриминант положителен для подстановки в каждую прямую, значит каждая прямая даёт две точки пересечения с окружностью. Эти точки не совпадают между прямыми, потому что общая точка прямых — только начало координат, а начало координат в этом случае окружности не принадлежит. Итого 2 + 2 = 4.
