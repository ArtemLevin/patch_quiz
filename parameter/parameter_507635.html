Наглядная математика с генерацией

Задача: найти все значения параметра a, при которых система имеет ровно одно решение.

⎧
⎨
⎩
|x + 2y + 1| ≤ 11,
(x − a)² + (y − 2a)² = 2 + a
Геометрический смысл.
Неравенство |x+2y+1| ≤ 11 задаёт полосу между прямыми x+2y=10 и x+2y=−12.
Уравнение (x−a)² + (y−2a)² = 2+a задаёт окружность с центром C(a,2a) и радиусом r = √(2+a).

1. Когда окружность вообще существует?

Нужно, чтобы 2+a ≥ 0, то есть a ≥ −2.

2. Удобно смотреть не на точки, а на величину x+2y

Для центра окружности C(a,2a) имеем
x_C + 2y_C = a + 2·(2a) = 5a.
Полоса задаётся условием
−12 ≤ x+2y ≤ 10.
Значит, окружность должна пересекать эту полосу ровно в одной точке.

3. Что значит «ровно одна точка»?

Если окружность заходит внутрь полосы хотя бы чуть-чуть, то обычно получается не одна точка, а целая дуга. Поэтому для невырожденной окружности ровно одна общая точка возможна только тогда, когда окружность касается одной из граничных прямых полосы и вся остальная окружность лежит вне полосы.

4. Проверим нижнюю границу x+2y=−12

Чтобы окружность касалась этой прямой «снаружи слева», надо было бы иметь
5a + √(5(2+a)) = −12.
Но при a ≥ −2 левая часть не может дать нужную конфигурацию: центр уже не уходит настолько далеко влево, чтобы вся окружность лежала вне полосы и касалась только линии x+2y=−12. Эта ветка решений не даёт.

5. Проверим верхнюю границу x+2y=10

Чтобы окружность была справа от полосы и касалась её в одной точке, нужно:
5a − √(5(2+a)) = 10.
Переносим корень:
√(5a+10) = 5a − 10,   поэтому   a ≥ 2.
Возводим в квадрат:
5a + 10 = (5a − 10)²
5a + 10 = 25a² − 100a + 100
25a² − 105a + 90 = 0
5a² − 21a + 18 = 0
D = 21² − 4·5·18 = 81
a = (21 ± 9)/10.
Получаем два корня:
a = 3,   a = 6/5.
Но условие a ≥ 2 оставляет только
a = 3.

6. Отдельно проверим вырожденный случай

При a = −2 радиус равен нулю, окружность превращается в единственную точку (−2,−4). Проверка полосы:
|−2 + 2·(−4) + 1| = |−9| = 9 ≤ 11.
Значит, система действительно имеет ровно одно решение.
Ответ: a = −2 и a = 3.
Короткая идея: нужно, чтобы проекция окружности на направление нормали к прямым x+2y=const пересекла отрезок [−12,10] ровно в одной точке. Это даёт либо вырождение a=−2, либо касание снаружи при a=3.
