Задача 1: Вычисление выражения

Условие: Вычислите 2√6 · cos(π/4) · sin(7π/6) · tg(−2π/3)

Показать решение
  1. Значения функций:
    • cos(π/4) = √2/2
    • sin(7π/6) = sin(π + π/6) = −sin(π/6) = −1/2
    • tg(−2π/3) = −tg(2π/3) = −(−√3) = √3
  2. Подстановка:
    2√6 · (√2/2) · (−1/2) · √3 =
    = − (2 · √6 · √2 · √3) / 4 =
    = − (2 · √36) / 4 = − (2 · 6) / 4 = −3
Ответ: −3

Задача 2: Уравнение с отбором

Условие: cos 2x − sin 2x = cos x + sin x + 1

а) Решить уравнение. б) Отбор на [−5π/2; −π].

Показать решение

а) Решение:

Используем формулы: cos 2x = cos²x − sin²x, 1 = sin²x + cos²x.

(cos²x − sin²x) − 2sin x cos x = cos x + sin x + (cos²x + sin²x)
−2sin²x − 2sin x cos x − cos x − sin x = 0
2sin x(sin x + cos x) + (sin x + cos x) = 0
(sin x + cos x)(2sin x + 1) = 0

Совокупность:

  • sin x + cos x = 0 ⇒ tg x = −1 ⇒ x = −π/4 + πn
  • sin x = −1/2 ⇒ x = −π/6 + 2πk, x = −5π/6 + 2πk

б) Отбор:

На отрезке [−2.5π; −π]:

  • Серия −π/4 + πn: при n=−2 получаем x = −9π/4.
  • Серия −π/6 + 2πk: при k=−1 получаем x = −13π/6.
  • Серия −5π/6 + 2πk: корней нет (−5π/6 > −π).
Ответ: −9π/4, −13π/6

Задача 4: Логарифмическое неравенство

Условие: 9·2^(log₃(5−x)) + 2^(1+log₃x) − 2^(log₃(5x−x²)) ≤ 18

Показать решение

1. ОДЗ: x ∈ (0; 5).

2. Упрощение: Заметим, что 5x − x² = x(5−x).

Обозначим a = 2^(log₃x), b = 2^(log₃(5−x)).

9b + 2a − ab ≤ 18
ab − 2a − 9b + 18 ≥ 0
(a − 9)(b − 2) ≥ 0

3. Анализ:

Так как x < 5, то log₃x < log₃5 < 2, значит a = 2^(log₃x) < 4.

Следовательно, (a − 9) всегда отрицательно.

Чтобы произведение было ≥ 0, второй множитель должен быть ≤ 0:

b − 2 ≤ 0 ⇒ 2^(log₃(5−x)) ≤ 2¹
log₃(5−x) ≤ 1
5 − x ≤ 3 ⇒ x ≥ 2

С учетом ОДЗ: x ∈ [2; 5).

Ответ: [2; 5)
