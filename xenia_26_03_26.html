Живой график функции

Функция: y = ((x − 3)(x² + 6x + 8)) / (x + 2)
Передвигай уровень m внизу страницы и наблюдай: у этой функции ровно одно пересечение с прямой y = m только при m = −10 и m = −49 / 4.

Интерактивный блок

Здесь можно увидеть каждую из трёх тем в действии: неравенство — на числовой прямой, среднюю скорость — в виде автоматического расчёта, а график функции — с управляемой прямой y = m.

неравенство

(7x − 5)² ≤ (5x − 7)²

(7x − 5)² − (5x − 7)² ≤ 0
((7x − 5) − (5x − 7)) · ((7x − 5) + (5x − 7)) ≤ 0
24(x + 1)(x − 1) ≤ 0
После деления на 24 остаётся (x + 1)(x − 1) ≤ 0, а значит ответ: x ∈ [−1; 1].
движение

Средняя скорость на равных частях пути

vср = Sобщ / tобщ
t = S / v
если части пути равны, средняя скорость = 2v₁v₂ / (v₁ + v₂)
88
км/ч — первая часть
72
км/ч — вторая часть
79,2
км/ч — итог
Нельзя брать обычное среднее арифметическое (88 + 72) / 2 = 80. Время на двух частях пути разное, поэтому нужно делить весь путь на всё время.
график и параметр

Функция с выколотой точкой

y = ((x − 3)(x² + 6x + 8)) / (x + 2)
x² + 6x + 8 = (x + 2)(x + 4)
y = (x − 3)(x + 4) = x² + x − 12, но x ≠ −2
График — это парабола y = x² + x − 12 с выколотой точкой (−2; −10). Ровно одно пересечение с y = m бывает при m = −10 и m = −49 / 4.

Средняя скорость на двух равных частях пути

Формула
vср = 2·88·72 / (88 + 72) = 79,2 км/ч
Проверка смысла
Ответ лежит между 72 км/ч и 88 км/ч, значит результат выглядит правдоподобно.

Теория в карточках

Короткие опорные идеи, которые чаще всего дают верный старт решению.

Квадратные неравенства

Ключевая формула
a² − b² = (a − b)(a + b)
Это одна из самых быстрых дорог от громоздкого квадратного выражения к разложению на множители.
Метод интервалов
Для произведения (x − α)(x − β) ≤ 0 ответ берётся между корнями, включая сами корни:
x ∈ [α; β]
Частая ошибка
Делить на число или выражение можно только после проверки знака. В задаче (7x − 5)² ≤ (5x − 7)² деление на 24 безопасно, потому что 24 > 0.

Средняя скорость

Главная формула
vср = Sобщ / tобщ
Средняя скорость — это не среднее арифметическое скоростей, а отношение всего пройденного пути ко всему времени.
Если части пути равны
Для двух равных участков пути со скоростями v₁ и v₂:
vср = 2v₁v₂ / (v₁ + v₂)
Это удобно как быстрый итог после табличного метода.
Почему 80 — неверно?
Число 80 получается как (88 + 72) / 2, но автомобиль движется на этих скоростях разное время. Значит, обычное среднее арифметическое здесь не работает.

График функции

Сначала ОДЗ
Для функции
y = ((x − 3)(x² + 6x + 8)) / (x + 2)
обязательно сначала записываем:
x ≠ −2
Сокращение дроби
Так как x² + 6x + 8 = (x + 2)(x + 4), получаем:
y = (x − 3)(x + 4) = x² + x − 12, но x ≠ −2
Это не обычная парабола, а парабола с выколотой точкой.
Ровно одна общая точка
Для прямой y = m есть два особых уровня:
m = −49 / 4 и m = −10
Первый касается вершины параболы, второй проходит через уровень выколотой точки.

Микро-алгоритмы

Неравенство
1) Перенести всё влево.
2) Применить a² − b².
3) Разложить на множители.
4) Построить числовую прямую.
5) Выбрать нужный промежуток.
Задача на движение
1) Обозначить равные части пути одной буквой.
2) Найти время на каждом участке: t = S / v.
3) Сложить пути и времена.
4) Использовать vср = Sобщ / tобщ.
Графо-аналитическая задача
1) Найти ОДЗ.
2) Упростить выражение.
3) Отдельно отметить удалённую точку.
4) Исследовать пересечения с y = m.

Разборы задач

Решения разбиты на короткие шаги, чтобы было видно не только ответ, но и логику переходов.

разбор 1

Неравенство (7x − 5)² ≤ (5x − 7)²

1
Переносим всё в левую часть: (7x − 5)² − (5x − 7)² ≤ 0.
2
Применяем формулу разности квадратов: ((7x − 5) − (5x − 7)) · ((7x − 5) + (5x − 7)) ≤ 0.
3
Упрощаем множители: (2x + 2)(12x − 12) ≤ 0.
4
Выносим общий множитель: 24(x + 1)(x − 1) ≤ 0.
5
Делим на 24 > 0: (x + 1)(x − 1) ≤ 0. Значит, x ∈ [−1; 1].
Ответ: x ∈ [−1; 1]
Можно было сразу раскрыть скобки: 49x² − 70x + 25 ≤ 25x² − 70x + 49 → 24x² − 24 ≤ 0 → x² ≤ 1 → x ∈ [−1; 1].
разбор 2

Средняя скорость: 88 км/ч и 72 км/ч

1
Пусть каждая половина пути равна x км. Тогда весь путь равен 2x км.
2
Время на первой половине: x / 88 ч. Время на второй половине: x / 72 ч.
3
Общее время: x / 88 + x / 72.
4
Средняя скорость: 2x / (x / 88 + x / 72).
5
После сокращения получаем 2 · 88 · 72 / (88 + 72) = 79,2 км/ч.
Ответ: 79,2 км/ч
80 — это среднее арифметическое скоростей, но не средняя скорость на всём пути. В задаче важны именно времена движения, а они на разных участках не совпадают.
разбор 3

График функции y = ((x − 3)(x² + 6x + 8)) / (x + 2) и значения m

1
ОДЗ: x ≠ −2.
2
Разлагаем квадратный трёхчлен: x² + 6x + 8 = (x + 2)(x + 4).
3
Сокращаем дробь: y = (x − 3)(x + 4) = x² + x − 12, но x ≠ −2.
4
При x = −2 получаем значение −10, значит на параболе выколота точка (−2; −10).
5
Вершина параболы: x = −1 / 2, y = −49 / 4.
6
Ровно одна общая точка с y = m получается либо в вершине, либо на уровне выколотой точки.
Ответ: m = −10 или m = −49 / 4
Парабола: y = x² + x − 12
Выколотая точка: (−2; −10)
Вершина: (−1 / 2; −49 / 4)
При m = −10 алгебраически уравнение x² + x − 12 = −10 имеет два решения: x = −2 и x = 1. Но точка x = −2 исключена из графика, поэтому остаётся только одно пересечение.
