Аналитическое решение
y = |x|·(x − 1) − 2x
Случай 1: x ≥ 0 Правая ветвь
|x| = x
y = x·(x − 1) − 2x
y = x² − x − 2x
y = x² − 3x

Парабола ветвями вверх, вершина в точке (1.5; −2.25)

Нули: x = 0 и x = 3

Случай 2: x < 0 Левая ветвь
|x| = −x
y = −x·(x − 1) − 2x
y = −x² + x − 2x
y = −x² − x

Парабола ветвями вниз, вершина в точке (−0.5; 0.25)

Нули: x = 0 и x = −1

(−0.5; 0.25)
