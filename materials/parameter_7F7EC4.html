Задача с параметром: ровно один корень на отрезке [0; 4]

√(7x − 4) · ln(x² − 8x + 17 − a²) = 0

Нужно найти все значения параметра a, при которых на отрезке [0; 4] уравнение имеет ровно один корень.

Настройки разбора

Цель: подготовка к ЕГЭ
Уровень: уверенный
Стиль: как репетитор
Фокус: геометрическая идея
Визуализация: координатная плоскость
Интерактивность: расширенная

Идея решения

1) Произведение равно нулю, когда хотя бы один множитель равен нулю, но при этом само выражение должно быть определено.
2) Первый множитель √(7x − 4) обращается в ноль только в точке x = 4/7.
3) Второй множитель ln(x² − 8x + 17 − a²) обращается в ноль, когда его аргумент равен 1. Это приводит к уравнению (x − 4)² = a².
4) Значит, мы считаем, сколько корней даёт точка x = 4/7 и сколько корней даёт пересечение параболы y = (x − 4)² + 1 с уровнем y = a² на допустимой части отрезка.

Подробное решение

Можно менять подробность и открывать шаги постепенно.
1
Что дано и где вообще можно искать корни
√(7x − 4) · ln(x² − 8x + 17 − a²) = 0, x ∈ [0; 4]
Сначала находим область определения: подкоренное выражение должно быть неотрицательным, а аргумент логарифма — положительным.

Чтобы выражение имело смысл, нужно одновременно:

7x − 4 ≥ 0  и  x² − 8x + 17 − a² > 0.

Из первого условия получаем:

x ≥ 4/7.

Так как по условию ещё задан отрезок [0; 4], реальные кандидаты на корни ищем только на отрезке:

x ∈ [4/7; 4].

Это важный шаг: значения x из части [0; 4/7) даже не рассматриваются как корни, потому что там √(7x − 4) не определён. Частая ошибка — сначала решить уравнение от произведения, а потом уже вспоминать про область определения. Здесь нужно наоборот: сначала разрешённые x, потом корни.

2
Когда нулём становится первый множитель
√(7x − 4) = 0  ⇔  7x − 4 = 0  ⇔  x = 4/7.
У первого множителя только одна точка зануления: x = 4/7.

Но точка x = 4/7 будет корнем исходного уравнения только тогда, когда логарифм в этой точке определён.

Подставим:

x² − 8x + 17 = (x − 4)² + 1, поэтому при x = 4/7 получаем ((4/7) − 4)² + 1 = (−24/7)² + 1 = 576/49 + 49/49 = 625/49.
Аргумент логарифма = 625/49 − a².
Нужно: 625/49 − a² > 0  ⇔  |a| < 25/7.

Здесь не требуется, чтобы логарифм был равен нулю. Достаточно только того, чтобы он существовал. Тогда произведение уже равно нулю из-за первого множителя.

3
Когда нулём становится логарифм
ln(x² − 8x + 17 − a²) = 0  ⇔  x² − 8x + 17 − a² = 1.
x² − 8x + 16 = a²  ⇔  (x − 4)² = a².
Получаем точки x = 4 ± |a|. На отрезке [0; 4] может остаться только x = 4 − |a|.

Решения уравнения (x − 4)² = a² таковы:

x = 4 − |a|  или  x = 4 + |a|.

Но на отрезке [0; 4] точка x = 4 + |a| не подходит, кроме случая a = 0, когда обе формулы дают одно и то же значение x = 4.

Значит, на нашем отрезке логарифм даёт кандидат:

x = 4 − |a|.

Но этот x должен ещё лежать в допустимой части, где корень существует:

4 − |a| ≥ 4/7  ⇔  |a| ≤ 24/7.

Геометрически это особенно удобно: мы смотрим, пересекает ли горизонтальная линия y = a² параболу y = (x − 4)² + 1 на разрешённой части графика. Левая точка пересечения имеет абсциссу x = 4 − |a|. Когда эта точка уходит левее x = 4/7, она перестаёт быть корнем исходного уравнения, потому что исходное выражение там не определено.

4
Считаем число корней по случаям
Нужно аккуратно сложить вклад точки x = 4/7 и вклад точки x = 4 − |a|.
Случай 1: |a| < 24/7
Точка x = 4/7 есть, и точка x = 4 − |a| тоже есть. Это две разные точки. Получаем 2 корня.
Случай 2: |a| = 24/7
Тогда x = 4 − |a| = 4/7. Оба механизма дают одну и ту же точку. Получаем ровно 1 корень.
Случай 3: 24/7 < |a| < 25/7
Логарифмический корень уже ушёл левее допустимой области, но точка x = 4/7 ещё остаётся допустимой. Получаем ровно 1 корень.
Случай 4: |a| ≥ 25/7
В точке x = 4/7 логарифм уже не определён, а логарифмического корня в допустимой области нет. Получаем 0 корней.

Обратите внимание на границу |a| = 25/7. При ней аргумент логарифма в точке x = 4/7 равен нулю, а логарифм нуля не существует. Поэтому эта граница не включается.

5
Ответ, проверка и окончательный вывод
24/7 ≤ |a| < 25/7.
Значит, a ∈ (−25/7; −24/7] ∪ [24/7; 25/7).

Почему эти значения подходят: при них либо две потенциальные точки сливаются в одну, либо остаётся только точка x = 4/7.

Почему других значений нет: при меньших |a| получается две разные точки, а при слишком больших |a| корней не остаётся вообще.

Итак, вся задача сводится к очень наглядной картине: есть фиксированная особая точка x = 4/7 и подвижная точка x = 4 − |a|. Нужно добиться, чтобы в допустимой области на отрезке осталось ровно одно значение x. Это и даёт ответ.

Типичные ошибки

1. Забыть про область определения
Нельзя считать корнем точку, где исходное выражение не определено. Это особенно важно для части x < 4/7.
2. Считать, что при произведении = 0 достаточно решить по множителям без проверки
Проверка существования логарифма обязательна.
3. Потерять точку x = 4/7
Она возникает не из логарифма, а из квадратного корня.
4. Включить границу |a| = 25/7
При ней аргумент логарифма в точке x = 4/7 равен нулю, а ln(0) не существует.

Мини-квиз

Вопрос 1. Почему на отрезке [0; 4] из решений (x − 4)² = a² обычно остаётся только x = 4 − |a|?
Вопрос 2. При каком условии точка x = 4/7 является корнем исходного уравнения?

Самопроверка

Проверь себя на примере: возьми a = 3.5. Сколько корней будет?
Показать ответ

|a| = 3.5 = 24.5/7, то есть 24/7 < |a| < 25/7. Значит, остаётся только точка x = 4/7. Ответ: 1 корень.

Проверь себя на примере: возьми a = 3. Тогда сколько корней?
Показать ответ

|a| = 3 < 24/7. Значит, есть и x = 4/7, и x = 4 − 3 = 1. Это две разные точки. Ответ: 2 корня.

Генератор похожих задач

Генератор сохраняет тот же метод: один корень идёт из квадратного корня, другой — из условия ln(...) = 0 через пересечение параболы и уровня a².

Печатная выжимка

Ответ: a ∈ (−25/7; −24/7] ∪ [24/7; 25/7).

Смысл: точка x = 4/7 существует при |a| < 25/7, а логарифмический корень x = 4 − |a| попадает в допустимую область при |a| ≤ 24/7. Нужно, чтобы в сумме остался ровно один корень.

Подсказки по чтению модели

1) Левую границу допустимых x задаёт вертикальная линия x = 4/7.
2) Горизонталь y = a² поднимается или опускается при изменении параметра.
3) Левое пересечение с параболой — это кандидат x = 4 − |a|.
4) Если оно совпало с x = 4/7 или ушло левее, число корней меняется.
Эта страница специально построена через геометрическую идею: вместо громоздких вычислений мы следим за тем, где находится уровень y = a² относительно параболы y = (x − 4)² + 1 и допустимой границы x = 4/7.
