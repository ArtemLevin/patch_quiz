Задача с параметром

√(x + 2a) · ln(x − a) = (x − 1) · ln(x − a),   x ∈ [0; 1]
Найти все значения параметра a, при которых уравнение имеет ровно один корень на отрезке [0; 1].
только школьные методы
геометрическая идея
пошаговое раскрытие
расширенная интерактивность

Итоговый ответ

a ∈ [−1/3; 0] ∪ {−1/2}
Ниже показано, откуда берётся именно этот набор: сначала выделяем общий множитель ln(x − a), затем проверяем область определения и отдельно считаем корни от каждого множителя.

Идея решения

  • Вынести общий множитель ln(x − a) и свести задачу к произведению двух множителей.
  • От множителя ln(x − a) получить кандидата x = a + 1, но не забыть про область определения: нужно ещё x + 2a ≥ 0.
  • От второго множителя √(x + 2a) = x − 1 на отрезке [0;1] возможен только крайний случай x = 1.
  • Потом проверить, когда получается ровно один корень, а когда корней нет или они совпадают/не совпадают.

Пошаговый разбор

1
Что дано
▾

Рассматриваем уравнение на отрезке [0;1]:

√(x + 2a) · ln(x − a) = (x − 1) · ln(x − a).

Нужно подобрать все a, при которых на этом отрезке будет ровно один корень.

Главная техническая тонкость: даже если ln(x − a)=0, исходное уравнение всё равно требует существования √(x+2a). Поэтому область определения проверяем для каждого найденного кандидата.
2
Что означает каждое условие
▸

Чтобы выражение было определено, нужны условия:

  • x − a > 0
  • x + 2a ≥ 0
  • x ∈ [0;1]

То есть каждый корень должен лежать сразу в трёх ограничениях: внутри отрезка, в области логарифма и в области корня.

3
Алгебраическая и графическая интерпретация
▸

Перенесём всё в одну сторону и вынесем общий множитель:

[√(x + 2a) − (x − 1)] · ln(x − a) = 0.

Значит, корни могут появиться из двух ветвей:

  1. ln(x − a) = 0
  2. √(x + 2a) = x − 1

Справа на рисунке эти две идеи показаны отдельно: вертикальный маркер x=a+1 для логарифмической ветви и графики y=√(x+2a), y=x−1 для второй ветви.

4
Ключевая идея
▸

На отрезке [0;1] величина x−1 ≤ 0, а корень √(x+2a) ≥ 0. Значит, уравнение

√(x + 2a) = x − 1

может выполняться только тогда, когда обе части равны нулю.

Это сразу сильно упрощает задачу: почти все корни идут из ветви ln(x−a)=0, а вторая ветвь даёт лишь один особый случай.

5
Пошаговое доведение до ответа
▸

Ветвь 1. Логарифм равен нулю

ln(x − a)=0 ⇔ x − a = 1 ⇔ x = a + 1.

Теперь проверяем все требования:

  • x ∈ [0;1] ⇒ a+1 ∈ [0;1] ⇒ a ∈ [−1;0]
  • x + 2a ≥ 0 ⇒ (a+1)+2a ≥ 0 ⇒ 3a+1 ≥ 0 ⇒ a ≥ −1/3

Совмещаем:

ветвь 1 даёт один корень тогда и только тогда, когда a ∈ [−1/3;0].

При этом сам корень равен x=a+1.

Ветвь 2. Равенство √(x + 2a) = x − 1

На отрезке [0;1] правая часть не положительна, левая не отрицательна. Поэтому нужно:

x−1 = 0 и √(x+2a)=0.

Отсюда:

x = 1,   1 + 2a = 0,   a = −1/2.

ветвь 2 даёт единственный специальный корень: x=1 при a=−1/2.

Сборка

Получили два набора параметров:

  • a ∈ [−1/3;0] — корень из логарифмической ветви;
  • a = −1/2 — особый корень x=1 из второй ветви.

Они не пересекаются, потому что −1/2 ∉ [−1/3;0].

Итак, ответ: a ∈ [−1/3;0] ∪ {−1/2}.

6
Явная проверка ответа
▸

Почему подходят все a ∈ [−1/3;0]?

  • x=a+1 ∈ [2/3;1]
  • x−a = 1 ⇒ ln(x−a)=0
  • x+2a = 3a+1 ≥ 0
  • вторая ветвь новых корней не даёт, потому что на [0;1] она возможна только в особой точке a=-1/2

Почему подходит a=-1/2?

  • x=1
  • √(1+2a)=√0=0
  • x−1=0
  • ln(1−a)=ln(3/2)
  • обе части равны нулю, значит это корень
  • логарифмическая ветвь не даёт корня, так как x=a+1=1/2, но там x+2a = -1/2 < 0, то есть исходное выражение не определено

Почему других значений нет?

Потому что вне этих случаев:

  • логарифмическая ветвь либо не попадает в отрезок, либо нарушает условие x+2a ≥ 0;
  • вторая ветвь невозможна, так как требует одновременно x=1 и a=-1/2.
7
Краткий вывод
▸

После разложения на множители задача сводится к двум очень простым источникам корней. Основной корень идёт из ln(x−a)=0, но проходит только при a ≥ −1/3. Ещё есть один специальный параметр a=-1/2, когда второй множитель зануляется в точке x=1.

Ответ: a ∈ [−1/3;0] ∪ {−1/2}.

Типичные ошибки

  • Считать, что из ln(x−a)=0 сразу следует корень x=a+1, не проверяя существование √(x+2a).
  • После факторизации забыть, что область определения исходного уравнения важнее, чем область определения уже преобразованной записи.
  • Пытаться квадратировать уравнение √(x+2a)=x−1, не заметив знаковую идею: на [0;1] правая часть неположительна.
  • Потерять специальный случай a=-1/2, когда обе части второго множителя становятся нулём.
  • Записать ответ только как [−1/3;0] и забыть отдельную точку −1/2.

Мини-квиз

1. Почему уравнение √(x+2a)=x−1 на отрезке [0;1] почти всегда невозможно?
Потому что левая часть неотрицательна, а правая на этом отрезке неположительна. Значит, равенство возможно только при одновременном равенстве обеих частей нулю.
2. Почему при a=-1/2 точка x=1/2 не считается корнем ветви ln(x−a)=0?
Потому что в исходном уравнении тогда нужно значение √(x+2a)=√(-1/2), а оно не определено.
3. Какая точка x соответствует логарифмической ветви?
Всегда x=a+1, если она попадает в отрезок и проходит область определения корня.

Самопроверка

Проверьте себя перед тем, как смотреть на итог
  • Вы отдельно выписали область определения x−a>0 и x+2a≥0?
  • Вы нашли обе ветви после вынесения общего множителя?
  • Вы заметили, что у второй ветви нужен именно край отрезка x=1?
  • Вы проверили, что для a=-1/2 логарифмическая ветвь не работает из-за области определения?
  • Вы объяснили, почему других параметров нет?
