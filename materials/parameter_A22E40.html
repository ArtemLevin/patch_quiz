A22E40 · Наглядная математика с генерацией

Решение задачи с параметром

√(2 − 3x) · ln(16x² − a²) = √(2 − 3x) · ln(4x + a)

Найти все значения параметра a, при которых уравнение имеет ровно один корень.

Цель: подготовка к ЕГЭ
Уровень: уверенный
Фокус: геометрическая идея
Метод: только школьные методы

Идея решения

  • Сначала выписываем область допустимых значений: нужны x ≤ 2/3, 16x² − a² > 0 и 4x + a > 0.
  • Множитель √(2 − 3x) может сам обращаться в ноль. Это даёт отдельный кандидат: x = 2/3.
  • Если x < 2/3, то корень не нулевой, поэтому можно приравнять логарифмы. Тогда задача сводится к простому квадратному уравнению, которое красиво раскладывается на множители.
  • В результате появляются ровно два возможных источника корней: горизонтальная линия x = 2/3 и наклонная линия x = (a + 1)/4 на плоскости (a, x).
  • Остаётся посмотреть, при каких a есть один источник, два источника или оба источника совпадают.

1

Что дано

Решаем уравнение

√(2 − 3x) · ln(16x² − a²) = √(2 − 3x) · ln(4x + a).

Нужно найти все значения параметра a, при которых уравнение имеет ровно один корень.

Нужно посчитать число корней уравнения в зависимости от параметра a.

2

Что означает каждое условие

Из корня получаем x ≤ 2/3.

Из логарифмов получаем две неравенства:

  • 16x² − a² > 0;
  • 4x + a > 0.
Важно. Нельзя сразу сокращать на √(2 − 3x), потому что этот множитель может равняться нулю при x = 2/3.

Здесь скрыта главная ловушка задачи: если бездумно поделить обе части на √(2 − 3x), то можно потерять корень x = 2/3. Поэтому разбираем уравнение по случаям.

ОДЗ: x ≤ 2/3, 16x² − a² > 0, 4x + a > 0.

3

Алгебраическая и графическая интерпретация

Уравнение распадается на два возможных механизма появления корня:

  • либо √(2 − 3x) = 0, то есть x = 2/3;
  • либо ln(16x² − a²) = ln(4x + a), то есть при положительных аргументах 16x² − a² = 4x + a.

На координатной плоскости (a, x) это превратится в горизонтальную прямую x = 2/3 и наклонную прямую x = (a + 1)/4.

Такой взгляд очень удобен: вместо сложного уравнения мы следим только за тем, сколько допустимых пересечений даёт вертикаль a = const с двумя кривыми-кандидатами на корни.

Два источника корней: x = 2/3 и x = (a + 1)/4.

4

Ключевая идея

При x < 2/3 множитель √(2 − 3x) положителен, значит можно приравнять логарифмы:

16x² − a² = 4x + a.

Переносим всё влево и раскладываем:

16x² − a² − 4x − a = 0,
(4x + a)(4x − a − 1) = 0.

Но на ОДЗ обязательно 4x + a > 0, значит множитель 4x + a не может равняться нулю.

Остаётся только 4x − a − 1 = 0, то есть x = (a + 1)/4.

Это и есть красивый момент задачи: один из множителей автоматически отбрасывается ОДЗ, поэтому из квадратного уравнения реально остаётся только одна ветвь.

После factorization: (4x + a)(4x − a − 1)=0, но первый множитель запрещён ОДЗ. Значит x=(a+1)/4.

5

Пошаговое доведение до ответа

Первый кандидат: x = 2/3.

Он подходит, если логарифмы определены:

4·(2/3) + a > 0  ⇔  a > −8/3,
16·(2/3)² − a² > 0  ⇔  64/9 − a² > 0  ⇔  |a| < 8/3.

Итак, x = 2/3 существует при −8/3 < a < 8/3.

Второй кандидат: x = (a + 1)/4.

Для него нужны условия:

  • 4x + a = 2a + 1 > 0  ⇔  a > −1/2;
  • x ≤ 2/3  ⇔  (a + 1)/4 ≤ 2/3  ⇔  a ≤ 5/3.

Значит второй корень существует при −1/2 < a ≤ 5/3.

Теперь считаем число корней:

  • при −8/3 < a ≤ −1/2 есть только горизонтальный корень x = 2/3;
  • при −1/2 < a < 5/3 есть два разных корня;
  • при a = 5/3 оба кандидата совпадают: (a + 1)/4 = 2/3;
  • при 5/3 < a < 8/3 снова остаётся только x = 2/3.
Совпадение корней в точке a = 5/3 — обязательный отдельный случай. Здесь корней не два, а один, потому что обе формулы дают одно и то же значение x = 2/3.

Первый корень существует при −8/3 < a < 8/3, второй — при −1/2 < a ≤ 5/3. Один корень получается на нужных кусках пересечения и вне пересечения.

6

Финальная проверка

Проверяем найденные промежутки:

  • Если a ∈ (−8/3, −1/2], второй кандидат не существует, а первый существует. Значит корень ровно один.
  • Если a ∈ [5/3, 8/3), то либо два кандидата совпадают в точке a = 5/3, либо остаётся только первый кандидат. Снова ровно один корень.
  • Внутри интервала (−1/2, 5/3) корней два.
  • Вне интервала (−8/3, 8/3) корней нет.

Ровно один корень только на двух кусках: слева до −1/2 и справа от 5/3, с учётом концов.

7

Краткий вывод

a ∈ (−8/3, −1/2] ∪ [5/3, 8/3)

Это и есть все значения параметра, при которых исходное уравнение имеет ровно один корень.

Типичные ошибки

  • Сразу сократить обе части на √(2 − 3x) и потерять случай x = 2/3.
  • После разложения на множители принять и случай 4x + a = 0, забыв, что тогда логарифм ln(4x + a) не определён.
  • Проверить только уравнение, но не проверить ОДЗ для найденных кандидатов.
  • В точке a = 5/3 посчитать два корня, хотя обе формулы дают одно и то же значение x = 2/3.
  • Забыть, что для логарифма нужен строгий знак > 0, а не ≥ 0.

Самопроверка

Генератор похожих задач

Генератор сохраняет тот же метод: отдельный корень от нулевого множителя и второй корень от равенства логарифмов.

Интерактивная модель

Схема показывает два возможных источника корней на плоскости (a, x): горизонталь x = 2/3 и диагональ x = (a + 1)/4. Вертикаль a = const показывает число допустимых пересечений.

Легенда обозначений

  • x = 2/3 — корень из-за нулевого множителя √(2 − 3x).
  • x = (a + 1)/4 — корень из равенства аргументов логарифмов.
  • Красные полосы на оси a — те значения параметра, где корень ровно один.

Проверка ответа без вычислительной рутины

На графике нужно просто следить, сколько допустимых пересечений имеет вертикаль a = const:
  • одно пересечение → ровно один корень;
  • два пересечения → два корня;
  • совпадение двух пересечений → снова один корень.
