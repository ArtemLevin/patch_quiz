9F53F5. Ровно один корень на отрезке [0; 1]

√(4x − 1) · ln(x² − 2x + 2 − a²) = 0
Цель: подготовка к ЕГЭ Уровень: уверенный Фокус: геометрическая идея Только школьные методы Пошаговое раскрытие

Идея решения

  1. Произведение равно нулю, когда √(4x − 1) = 0 или ln(...) = 0, но всё выражение должно быть определено.
  2. Первый источник корня даёт специальную точку x = 1/4.
  3. Второй источник корней даёт уравнение (x − 1)² = a², то есть точки вида x = 1 ± |a|.
  4. Дальше нужно понять, когда на отрезке [0;1] и с учётом области определения остаётся ровно одна точка.

Пошаговое решение

Можно раскрывать постепенно и менять подробность объяснения.
1) Что дано
Нужно найти все значения параметра a, при которых уравнение √(4x − 1) · ln(x² − 2x + 2 − a²) = 0 имеет ровно один корень на отрезке [0;1].
Здесь важно не просто решить уравнение, а именно посчитать число его корней на заданном отрезке.
2) Что означает каждое условие
Чтобы выражение существовало, должны выполняться два требования:
  • 4x − 1 ≥ 0, то есть x ≥ 1/4;
  • x² − 2x + 2 − a² > 0, потому что аргумент логарифма должен быть положительным.
На отрезке [0;1] первая часть сразу отсекает кусок [0;1/4). Значит, искать корни имеет смысл только на [1/4;1].
Даже если какой-то множитель «мог бы дать ноль», это не считается корнем, если всё выражение в этой точке не определено.
3) Интерпретация
Произведение равно нулю, если хотя бы один множитель равен нулю:
  • √(4x − 1) = 0;
  • ln(x² − 2x + 2 − a²) = 0.
Во втором случае ln T = 0 ⇔ T = 1. Поэтому графически удобно смотреть на параболу y = (x − 1)² и на горизонтальный уровень y = a².
4) Ключевая идея
Считаем отдельно:
  1. когда существует специальный корень x = 1/4;
  2. когда существуют корни от логарифма;
  3. не совпадают ли они.
Именно это и превращает задачу в задачу на число решений: надо учесть существование, попадание на отрезок и возможное совпадение корней.
5) Пошаговое доведение до ответа
Первый источник корня.
√(4x − 1) = 0 ⇔ 4x − 1 = 0 ⇔ x = 1/4.
Но точка x = 1/4 годится только тогда, когда логарифм определён:
(1/4)² − 2·(1/4) + 2 − a² > 0
1/16 − 1/2 + 2 − a² > 0
25/16 − a² > 0
|a| < 5/4.
Второй источник корней.
ln(x² − 2x + 2 − a²) = 0 ⇔ x² − 2x + 2 − a² = 1 ⇔ x² − 2x + 1 − a² = 0 ⇔ (x − 1)² = a².
Отсюда x = 1 ± |a|. На отрезке [0;1] точка 1 + |a| не подходит, остаётся только x = 1 − |a|, причём нужно, чтобы она лежала не левее 1/4:
1 − |a| ≥ 1/4 ⇔ |a| ≤ 3/4.
Здесь дополнительных ограничений на логарифм уже не нужно: при ln(...) = 0 его аргумент автоматически равен 1, а это положительное число.
Теперь считаем число корней.
  • Если |a| < 3/4, есть два разных корня: 1/4 и 1 − |a|.
  • Если |a| = 3/4, второй корень равен 1 − 3/4 = 1/4, то есть оба источника дают одну и ту же точку.
  • Если 3/4 < |a| < 5/4, логарифмический корень уже ушёл левее 1/4, но специальный корень 1/4 ещё существует. Значит, корень один.
  • Если |a| ≥ 5/4, точка 1/4 перестаёт входить в область определения, а логарифмического корня на допустимом участке нет. Корней нет.
6) Финальная проверка
Подходят случаи, когда получается ровно один корень:
|a| = 3/4
или
3/4 < |a| < 5/4.
Объединяем:
3/4 ≤ |a| < 5/4.
Почему других значений нет:
  • при |a| < 3/4 корней два;
  • при |a| ≥ 5/4 корней нет.
7) Краткий вывод
a ∈ (−5/4; −3/4] ∪ [3/4; 5/4)
То же самое можно записать короче: 3/4 ≤ |a| < 5/4.

Типичные ошибки

  • Забыть, что всё уравнение должно быть определено, и автоматически принять x = 1/4 при любых a.
  • После ln T = 0 написать T = 0 вместо правильного T = 1.
  • Взять обе точки 1 ± |a| на отрезке [0;1], хотя 1 + |a| почти всегда лежит правее 1.
  • Не заметить совпадение корней при |a| = 3/4.
  • Смешать условия «корень существует» и «корень лежит в допустимой области x ≥ 1/4».
