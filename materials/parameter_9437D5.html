Ровно один корень на отрезке [0; 1]

Разбираем уравнение школьным методом: раскладываем условие на два независимых источника корней, затем для каждого источника отдельно проверяем область определения и попадание в отрезок.
Фокус: геометрическая идея
Интерактивность: расширенная
Режим: пошаговое раскрытие
Автор: Лёвин Артём Александрович
(5x − 2) · ln(x + a) = (5x − 2) · ln(2x − a),    x ∈ [0; 1]

Идея решения

  1. Вынесем общий множитель 5x − 2 и получим произведение, равное нулю.
  2. Отсюда возможны только два типа корней: x = 2/5 и ln(x + a) = ln(2x − a).
  3. Второе равенство логарифмов даёт x = 2a, но только там, где оба логарифма определены.
  4. Дальше считаем не «формальные решения», а именно допустимые корни на [0; 1].

1Что дано

Нужно найти все значения параметра a, при которых уравнение имеет ровно один корень на отрезке [0; 1].
Здесь важно не просто решить уравнение, а посчитать количество допустимых корней на данном отрезке.
Типичная ошибка на этом месте: сразу приравнять аргументы логарифмов и забыть про множитель 5x − 2.

2Что означает условие

Переносим всё в одну сторону:
(5x − 2) · [ln(x + a) − ln(2x − a)] = 0.
Значит, корень может появиться либо из уравнения 5x − 2 = 0, либо из уравнения ln(x + a) = ln(2x − a).
Но оба варианта надо проверять только там, где логарифмы существуют: x + a > 0 и 2x − a > 0.

3Алгебраическая интерпретация первого источника корней

5x − 2 = 0  ⇒  x = 2/5.
Это «специальный» корень: он появляется из множителя 5x − 2 и не требует равенства логарифмов.
Чтобы x = 2/5 был допустим, нужно:
2/5 + a > 0  и  4/5 − a > 0.
Итак, корень x = 2/5 существует при  −2/5 < a < 4/5.
На правой модели этот корень отмечен голубой точкой на оси x.

4Алгебраическая интерпретация второго источника корней

ln(x + a) = ln(2x − a).
Логарифм — возрастающая функция, значит равенство логарифмов эквивалентно равенству аргументов.
x + a = 2x − a  ⇒  x = 2a.
Для этого корня ещё нужна ОДЗ: x + a > 0 и 2x − a > 0.
Подставляем x = 2a: получаем 3a > 0, то есть a > 0.
И ещё x = 2a должен лежать на [0; 1], значит 0 < a ≤ 1/2.
Итак, корень x = 2a существует при  0 < a ≤ 1/2.

5Ключевая идея подсчёта

У нас есть только два кандидата на корень:
x₁ = 2/5 и x₂ = 2a.
Теперь нужно добиться, чтобы на отрезке [0; 1] существовал ровно один допустимый корень.
Если оба корня существуют и различны, то корней уже два — это нам не подходит.

6Пошаговое доведение до ответа

1) Только корень x = 2/5:
Нужно, чтобы −2/5 < a < 4/5 и одновременно корень x = 2a не существовал.
Корень x = 2a существует при 0 < a ≤ 1/2, значит вне этого промежутка остаётся:
−2/5 < a ≤ 0 и 1/2 < a < 4/5.
2) Оба корня совпадают:
2a = 2/5  ⇒  a = 1/5.
При a = 1/5 оба «источника» дают один и тот же корень x = 2/5, поэтому фактически корень один.
Получаем кандидаты: (−2/5; 0] ∪ {1/5} ∪ (1/2; 4/5).

7Финальная проверка

Если a ∈ (−2/5; 0], существует только x = 2/5. Значит корень ровно один.
Если a = 1/5, оба способа дают x = 2/5. Значит корень ровно один.
Если a ∈ (1/2; 4/5), опять существует только x = 2/5. Значит корень ровно один.
Если 0 < a ≤ 1/2, a ≠ 1/5, то существуют два разных корня: x = 2/5 и x = 2a.
Если a ≤ −2/5 или a ≥ 4/5, корень x = 2/5 исчезает, а второй корень в этих зонах тоже не даёт ровно один корень.

8Краткий вывод

Ответ: a ∈ (−2/5; 0] ∪ {1/5} ∪ (1/2; 4/5).

Фиолетовые графики

Типичные ошибки

  • Сразу сократить на 5x − 2. Так делать нельзя: при x = 2/5 можно потерять корень.
  • Из ln(x + a) = ln(2x − a) получить x = 2a и забыть проверить ОДЗ.
  • Посчитать два «способа появления» корня как два разных корня даже при a = 1/5.
  • Проверять только существование корней, но не считать их количество именно на отрезке [0; 1].
  • Ошибиться со строгими знаками в условиях −2/5 < a < 4/5.
