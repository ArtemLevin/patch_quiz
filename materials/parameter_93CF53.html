Задача с параметром: ровно один корень на отрезке [0; 1]

Слева — прозрачный разбор по шагам, справа — интерактивная координатная модель. Можно менять параметр a, включать и выключать детали, масштабировать рисунок и применять похожие задачи к той же схеме.
√(x² − a²) = √(4x² − (4a + 2)x + 2a), x ∈ [0; 1].
Найти все значения a, при которых уравнение имеет ровно один корень.
ЦельПодготовка к ЕГЭ
ФокусГеометрическая идея + логика отбора случаев
РежимПошаговое раскрытие
МетодТолько школьные методы

Идея решения

  1. Приравниваем подкоренные выражения.
  2. Получаем квадратное уравнение, которое красиво раскладывается на множители.
  3. Появляются два кандидата на корень: x = a и x = (a + 2)/3.
  4. Для каждого кандидата проверяем два пункта: попадание в [0;1] и неотрицательность радикандов.
  5. После этого уже легко определить, когда корень ровно один.

Подробное решение

1) Что дано

Исходное уравнение и рабочий отрезок

Нужно выяснить, при каких значениях параметра a уравнение

√(x² − a²) = √(4x² − (4a + 2)x + 2a)

имеет ровно один корень на отрезке [0;1].

Сначала найдём все возможные кандидаты на корень, а потом отсеем лишние.

Так как обе части — квадратные корни, сначала удобно приравнять подкоренные выражения.

Это типичная задача на параметр: сами кандидаты на корни рождаются из алгебры, а окончательный ответ — из проверки условий допустимости.

2) Что означает условие

Равенство двух квадратных корней

Если √U = √V, то обязательно U = V, а также U ≥ 0 и V ≥ 0.

Поэтому достаточно решить

x² − a² = 4x² − (4a + 2)x + 2a

и затем проверить, какие найденные значения x действительно допустимы для исходного уравнения.

Ключевая идея: сначала находим корни-кандидаты, потом проверяем, кто из них реально живёт в исходной задаче.
3) Алгебраическая интерпретация

Переносим всё в одну сторону и раскладываем на множители

x² − a² = 4x² − (4a + 2)x + 2a
3x² − (4a + 2)x + a² + 2a = 0

Теперь замечаем разложение:

3x² − (4a + 2)x + a² + 2a = (x − a)(3x − a − 2).

Значит, возможны только два кандидата:

x = a   или   x = (a + 2)/3.

Проверка разложения прямым умножением полностью проходит: коэффициенты совпадают.

4) Ключевая идея

Первый кандидат: x = a

Подставляем x = a в левый радиканд:

x² − a² = a² − a² = 0.

Значит, при x = a оба корня равны нулю. Осталось проверить, лежит ли эта точка на отрезке [0;1].

x = a ∈ [0;1] ⇔ 0 ≤ a ≤ 1.
Итак, кандидат x = a даёт корень при a ∈ [0;1].
5) Пошаговое доведение

Второй кандидат: x = (a + 2)/3

Сначала проверим попадание в рабочий отрезок:

0 ≤ (a + 2)/3 ≤ 1 ⇔ −2 ≤ a ≤ 1.

Но этого мало: нужно ещё, чтобы подкоренные выражения были неотрицательны. Достаточно проверить левый радиканд:

x² − a² = ((a + 2)/3)² − a² = (−8a² + 4a + 4)/9.
(−8a² + 4a + 4)/9 ≥ 0 ⇔ 2a² − a − 1 ≤ 0.
2a² − a − 1 = (2a + 1)(a − 1).
(2a + 1)(a − 1) ≤ 0 ⇔ −1/2 ≤ a ≤ 1.

Это уже сильнее, чем условие −2 ≤ a ≤ 1.

Значит, кандидат x = (a + 2)/3 даёт корень при a ∈ [−1/2;1].
6) Считаем количество корней

Когда получается ровно один корень?

Кандидат x = a

существует при 0 ≤ a ≤ 1

Кандидат x = (a + 2)/3

существует при −1/2 ≤ a ≤ 1

  • a < −1/2: корней нет.
  • −1/2 ≤ a < 0: первый кандидат не подходит, второй подходит → ровно один корень.
  • 0 ≤ a < 1: оба кандидата подходят и различны → два корня.
  • a = 1: кандидаты совпадают, потому что a = (a + 2)/3, то есть снова один корень x = 1.
  • a > 1: корней нет.

Совпадение двух кандидатов происходит только при 3a = a + 2, то есть при a = 1.

7) Финальная проверка и вывод

Ответ

a ∈ [−1/2; 0) ∪ {1}.
Проверка завершена:
• для a ∈ [−1/2;0) существует только корень x = (a + 2)/3;
• при a = 1 два кандидата сливаются в одну точку x = 1;
• во всех остальных случаях корней либо нет, либо их два.

Типичные ошибки

  • После приравнивания радикандов забыть про неотрицательность подкоренных выражений.
  • Считать, что любой корень уравнения (x − a)(3x − a − 2) = 0 автоматически подходит исходному уравнению.
  • Не проверить отдельно, что x = a должен лежать в отрезке [0;1].
  • На промежутке 0 ≤ a < 1 ошибочно записать один корень, хотя там их два.
  • Пропустить особую точку a = 1, где два кандидата совпадают.

Самопроверка

  • Почему из равенства двух квадратных корней можно перейти к равенству подкоренных выражений?
  • Почему кандидат x = a автоматически даёт нулевой радиканд?
  • Почему для кандидата x = (a + 2)/3 нужно проверять знак радиканда, а не только отрезок?
  • Почему значение a = 1 рассматривается отдельно?
