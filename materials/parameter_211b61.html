211B61. Найдите все значения a, при которых уравнение имеет единственное решение

Задача. (2x + a + 1 + tan x)2 = (2x + a − 1 − tan x)2,   x ∈ [−π/2; π/2]
Важно: функция tan x не определена при x = ±π/2, поэтому реально рассматриваем только точки из интервала (−π/2; π/2).
Цель: подготовка к ЕГЭ Уровень: уверенный Фокус: геометрическая идея Режим: пошаговое раскрытие Автор: Лёвин Артём Александрович

Идея решения

  1. Используем факт: из равенства квадратов U2 = V2 следует U = V или U = −V.
  2. После этого исходное уравнение распадается на две независимые ветки: одна даёт фиксированное решение по x, другая — решение, зависящее от параметра a.
  3. Дальше нужно не просто найти обе ветки, а понять, когда на допустимом промежутке остаётся ровно одна точка.
  4. На рисунке справа это видно как сочетание постоянной точки x = −π/4 и подвижной точки x = −a/2.
Ключевой замысел: задача на параметр здесь превращается в аккуратный подсчёт числа корней после разбиения на две простые ветви.

Пошаговое решение

Так как в уравнении есть tan x, точки x = −π/2 и x = π/2 не подходят. Значит, искать решения нужно только при x ∈ (−π/2; π/2).
На отрезке [−π/2; π/2] записано условие задачи, но функция tan x не определена в концах. Поэтому фактическая область поиска решений такая:
x ∈ (−π/2; π/2).
Формально задача говорит про отрезок [−π/2; π/2], но у нас в формуле присутствует tan x. А функция tan x не существует при x = π/2 + πk, в частности при x = −π/2 и x = π/2. Поэтому конечные точки отрезка сразу исключаются из рассмотрения. Это очень важный шаг: если забыть про него, можно ошибочно засчитать «корни» на границе.
ОДЗ: x ∈ (−π/2; π/2).
Нельзя подставлять x = ±π/2 даже для проверки: исходное выражение там не определено.
Если U2 = V2, то U = V или U = −V.
Обозначим
U = 2x + a + 1 + tan x,   V = 2x + a − 1 − tan x.
Тогда из равенства квадратов получаем два случая:
{
U = V
U = −V
Используем стандартный факт: числа имеют равные квадраты тогда и только тогда, когда они либо равны, либо противоположны. Поэтому вместо одного уравнения четвёртой «снаружи» формы мы получаем два гораздо более простых уравнения. Это и есть переломный момент решения.
(2x + a + 1 + tan x)2 = (2x + a − 1 − tan x)2
⇔ 2x + a + 1 + tan x = 2x + a − 1 − tan x   или   2x + a + 1 + tan x = −(2x + a − 1 − tan x).
Получаем 1 + tan x = −1 − tan x, то есть tan x = −1. Значит, x = −π/4.
Приравниваем левые части:
2x + a + 1 + tan x = 2x + a − 1 − tan x.
Сокращаем одинаковые части:
1 + tan x = −1 − tan x.
Тогда
2 tan x = −2,   tan x = −1.
На интервале (−π/2; π/2) это даёт единственную точку
x = −π/4.
В первом случае параметр a полностью исчезает — это очень удобно. После сокращения 2x + a остаётся тригонометрическое уравнение:
1 + tan x = −1 − tan x.
Переносим tan x влево, числа вправо:
tan x + tan x = −1 − 1.
2 tan x = −2.
tan x = −1.
На промежутке (−π/2; π/2) функция tan x строго возрастает и принимает каждое значение ровно один раз. Значению −1 соответствует угол −π/4. Значит, первая ветка всегда даёт один и тот же корень:
x1 = −π/4.
Этот корень существует при любом a.
Получаем 4x + 2a = 0, значит x = −a/2.
Рассмотрим второй случай:
2x + a + 1 + tan x = −(2x + a − 1 − tan x).
Раскрываем скобки:
2x + a + 1 + tan x = −2x − a + 1 + tan x.
Сокращаем одинаковые слагаемые 1 и tan x:
2x + a = −2x − a.
4x + 2a = 0.
x = −a/2.
Во второй ветке как раз появляется зависимость от параметра. После раскрытия скобок получается приятное линейное уравнение:
2x + a + 1 + tan x = −2x − a + 1 + tan x.
Здесь сокращаются и единицы, и tan x. Это значит, что тригонометрии в этой ветке уже не остаётся вовсе:
2x + a = −2x − a.
4x + 2a = 0.
x2 = −a/2.
Но этот корень надо ещё проверить на принадлежность ОДЗ, то есть он годится только тогда, когда попадает внутрь интервала (−π/2; π/2).
Всегда есть x = −π/4. Второй корень x = −a/2 либо не попадает в ОДЗ, либо совпадает с −π/4. В этих случаях решение единственно.
Итак, кандидаты на решения:
x1 = −π/4,   x2 = −a/2.
Первый корень есть всегда. Чтобы решение было единственным, второй корень должен:
  • либо не принадлежать интервалу (−π/2; π/2);
  • либо совпасть с x = −π/4.
Условие непопадания в ОДЗ:
−a/2 ∉ (−π/2; π/2) ⇔ a ∉ (−π; π).
Условие совпадения:
−a/2 = −π/4 ⇔ a = π/2.
После разбиения всё стало очень прозрачным:
x1 = −π/4 — существует всегда;
x2 = −a/2 — существует только если попадает в ОДЗ.
Значит, ровно одно решение будет в двух ситуациях.
  1. Вторая ветка не даёт допустимого корня.
    Для этого нужно, чтобы x = −a/2 не лежал в интервале (−π/2; π/2):
    −a/2 ≤ −π/2   или   −a/2 ≥ π/2.
    Умножаем на −2 и не забываем поменять знаки:
    a ≥ π   или   a ≤ −π.
    То есть
    a ∈ (−∞; −π] ∪ [π; +∞).
  2. Вторая ветка попадает ровно в ту же точку, что и первая.
    Тогда два найденных «кандидата» совпадают:
    −a/2 = −π/4.
    a = π/2.
Больше вариантов нет: если a ∈ (−π; π) и a ≠ π/2, то второй корень существует и отличается от −π/4, значит решений будет уже два.
Итак, единственность достигается либо при выпадении второй ветки из ОДЗ, либо при совпадении двух ветвей в одной точке.
Если a ≤ −π или a ≥ π, второй корень не входит в ОДЗ, остаётся только x = −π/4. Если a = π/2, оба корня совпадают. В остальных случаях корней два.
Проверим три случая:
  1. a ≤ −π или a ≥ π. Тогда x = −a/2 не лежит в (−π/2; π/2), значит из второй ветки допустимого корня нет. Остаётся только x = −π/4.
  2. a = π/2. Тогда x = −a/2 = −π/4. Обе ветки дают одну и ту же точку, значит решение одно.
  3. a ∈ (−π; π), a ≠ π/2. Тогда x = −a/2 — допустимый корень и он не равен −π/4. Значит, решений два.
Проверка обязана разделять «подходит» и «не подходит».
  1. При a ∈ (−∞; −π] ∪ [π; +∞) число x = −a/2 находится не внутри (−π/2; π/2), а значит, во второй ветке допустимого решения нет. Но первая ветка по-прежнему даёт x = −π/4. Следовательно, имеется ровно один допустимый корень.
  2. При a = π/2 получаем x2 = −a/2 = −π/4 = x1. Ветки не дают два разных решения — они дают одну и ту же точку. Значит, решение опять единственно.
  3. Если a лежит внутри интервала (−π; π), но не равно π/2, то x2 = −a/2 принадлежит ОДЗ и отличается от −π/4. Тогда существуют две разные допустимые точки, то есть условие задачи нарушается.
Этим доказано не только то, что найденные значения подходят, но и то, что никаких других значений параметра быть не может.
Ответ: a ∈ (−∞; −π] ∪ {π/2} ∪ [π; +∞).
  • После разложения равенства квадратов получаем два корня-кандидата: x = −π/4 и x = −a/2.
  • Первый корень существует всегда, второй — только при попадании в ОДЗ.
  • Ровно одно решение получается, если второй корень отсутствует или совпадает с первым.
Итог: a ≤ −π, либо a = π/2, либо a ≥ π.

Типичные ошибки

  • Считать, что на концах x = ±π/2 можно проверять уравнение. Нельзя: там tan x не существует.
  • Из U2 = V2 делать только вывод U = V и забывать про случай U = −V.
  • После получения x = −a/2 не проверить, попадает ли этот x в ОДЗ.
  • Не заметить случай совпадения двух веток: x = −a/2 = −π/4, то есть a = π/2.
  • Перепутать знак при переходе от условия −a/2 ∉ (−π/2; π/2) к условию на параметр a.
