3E3293 · Задача с параметром

Ровно один корень на отрезке [0; 3]

Наглядный разбор уравнения с параметром школьными методами: отдельно отслеживаем, когда нуль даёт логарифм, а когда — квадратный корень, и затем считаем количество различных корней на отрезке.
Цель: подготовка к ЕГЭ
Уровень: уверенный
Фокус: геометрическая идея
Интерактивность: расширенная
ln(4x − 1) · √(x² − 6x + 6a − a²) = 0,   x ∈ [0; 3]
Автор: Лёвин Артём Александрович

Идея решения

  • У произведения двух множителей нуль получается тогда и только тогда, когда хотя бы один множитель равен нулю, но оба множителя при этом должны быть определены.
  • Логарифм даёт ровно одну кандидатуру: x = 1/2.
  • Подкоренное выражение удобно разложить как (x − 3)² − (a − 3)², поэтому нули корня сразу видны: x = a и x = 6 − a.
  • Дальше остаётся аккуратно посчитать, в какие моменты среди точек 1/2, a, 6 − a на отрезке [0; 3] остаётся ровно одна.

Режим подробности

При переключении меняется объём пояснений, но сама математическая логика и ответ остаются теми же.

Пошаговое решение

1) Что дано

Нужно найти все значения параметра a, при которых уравнение

ln(4x − 1) · √(x² − 6x + 6a − a²) = 0

имеет на отрезке [0; 3] ровно один корень.

Главная мысль: сначала ищем все возможные корни, потом считаем их число.

Поскольку слева стоит произведение, корни будут появляться из двух источников:

  • из равенства ln(4x − 1) = 0;
  • из равенства √(x² − 6x + 6a − a²) = 0.

Но нельзя забывать и про область допустимых значений: логарифм должен быть определён, а подкоренное выражение не может быть отрицательным. В этой задаче очень важно не только найти «формальные» точки, но и проверить, действительно ли они допустимы.

2) Что означает каждое условие

Логарифм определён, если 4x − 1 > 0, то есть x > 1/4.

Квадратный корень определён, если x² − 6x + 6a − a² ≥ 0.

Значит, каждый корень обязан лежать не просто на [0; 3], а на части этого отрезка, где одновременно выполнены оба условия.

Уже отсюда видно, что точка x = 1/4 не подходит: логарифм в ней не определён.

Это один из самых частых источников ошибок: ученик находит нуль подкоренного выражения в точке x = 1/4 и автоматически засчитывает его как корень всего уравнения. Так делать нельзя, потому что множитель ln(4x − 1) в этой точке вообще не существует.

3) Интерпретация логарифмического множителя

Решаем ln(4x − 1) = 0.

Логарифм равен нулю тогда и только тогда, когда его аргумент равен 1:

4x − 1 = 1 ⇒ 4x = 2 ⇒ x = 1/2.

Значит, от логарифма возникает одна кандидатура: x = 1/2.

Но эта точка станет настоящим корнем только тогда, когда подкоренное выражение в ней неотрицательно.

Подставим x = 1/2:

(1/2)² − 6·(1/2) + 6a − a² = 1/4 − 3 + 6a − a² = −a² + 6a − 11/4.

Требуем неотрицательность:

−a² + 6a − 11/4 ≥ 0 ⇔ a² − 6a + 11/4 ≤ 0.

Корни квадратного трёхчлена:

D = 36 − 11 = 25,   a₁ = 1/2,   a₂ = 11/2.

Следовательно, точка x = 1/2 существует как корень при a ∈ [1/2; 11/2].

4) Интерпретация корневого множителя

Теперь ищем нули подкоренного выражения:

x² − 6x + 6a − a² = 0.

Выделяем квадрат:

x² − 6x + 9 − 9 + 6a − a² = 0

(x − 3)² − (a − 3)² = 0.

Разность квадратов равна нулю, значит:

x − 3 = ±(a − 3).

Отсюда получаем две кандидатуры:

x = a и x = 6 − a.

Это очень удобная геометрическая картина: две симметричные точки относительно числа 3.

Но обе эти точки надо ещё проверить на попадание в область допустимых значений для логарифма и на отрезок [0; 3].

  • x = a подходит как корень тогда и только тогда, когда a ∈ (1/4; 3].
  • x = 6 − a подходит как корень тогда и только тогда, когда 6 − a ∈ (1/4; 3], то есть a ∈ [3; 23/4).

5) Ключевая идея: считаем точки 1/2, a, 6 − a

Все возможные корни — это точки:

x = 1/2,   x = a,   x = 6 − a,

но засчитываются только те, которые действительно допустимы.

Дальше идём по промежуткам параметра a и смотрим, сколько точек остаётся.

Надо помнить о совпадениях:

  • x = a и x = 6 − a совпадают при a = 3;
  • x = 1/2 и x = a совпадают при a = 1/2;
  • x = 1/2 и x = 6 − a совпадают при a = 11/2.

Именно из-за этих совпадений ответ нельзя получить простым «объединением условий». Нужно считать количество различных корней, а не количество формул, которые дали нуль.

6) Разбор по случаям

a ≤ 1/4 Корней нет: x = a не проходит условие x > 1/4, x = 6 − a не попадает в [0; 3], x = 1/2 ещё не появился.
1/4 < a < 1/2 Есть только корень x = a. Следовательно, корень ровно один.
a = 1/2 Точки x = a и x = 1/2 совпали. Всё равно получаем один корень.
1/2 < a < 3 Есть два различных корня: x = 1/2 и x = a.
a = 3 Получаются два корня: x = 1/2 и x = 3.
3 < a < 11/2 Есть два различных корня: x = 1/2 и x = 6 − a.
a = 11/2 Точки x = 1/2 и x = 6 − a совпали. Получаем один корень.
11/2 < a < 23/4 Остаётся только корень x = 6 − a. Значит, корень ровно один.
a ≥ 23/4 Корней нет. При a = 23/4 формально x = 6 − a = 1/4, но логарифм там не определён.

7) Финальная проверка и ответ

Ответ: a ∈ (1/4; 1/2] ∪ [11/2; 23/4).

Проверка:

  • при a ∈ (1/4; 1/2) единственный корень — x = a;
  • при a = 1/2 единственный корень — x = 1/2;
  • при a = 11/2 единственный корень — x = 1/2 = 6 − a;
  • при a ∈ (11/2; 23/4) единственный корень — x = 6 − a.

Во всех остальных случаях либо корней нет, либо их два.

Печатная версия: ответ a ∈ (1/4; 1/2] ∪ [11/2; 23/4).

Типичные ошибки

  • Засчитать x = 1/4 как корень. Нельзя: ln(4x − 1) в этой точке не определён.
  • Не проверить x = 1/2. Логарифм даёт эту точку всегда формально, но настоящим корнем она становится только при a ∈ [1/2; 11/2].
  • Забыть про совпадения корней. При a = 1/2 и a = 11/2 число формул-источников больше, чем число различных корней.
  • Не заметить симметрию x = a и x = 6 − a. Из-за неё задача решается намного быстрее и чище.
  • Считать только нули множителей без ОДЗ. Для произведения этого недостаточно: оба множителя должны быть определены.
