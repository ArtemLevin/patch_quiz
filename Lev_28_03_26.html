Меняйте цифры a и b. Система автоматически строит число, число после перестановки, сумму квадратов и проверяет условие b = a + 2.

Если b = a + 2, то n = 11a + 2, m = 11a + 20.

a² + 2a − 3 = 0
a = 1, b = 3, число = 13

3 = 1 + 2 ✓

Теория

Базовые правила, которые нужны для задач на цифры числа.
Если первая цифра равна a, а вторая цифра равна b, то двузначное число записывается так:
n = 10a + b
  • a — цифра десятков, поэтому a не может быть равна 0.
  • b — цифра единиц, поэтому b может быть любой цифрой от 0 до 9.
  • Неверно писать ab = a + b. Верно только 10a + b.
После перестановки цифр число 10a + b превращается в:
m = 10b + a
  • Например, из 47 получается 74.
  • Разность этих чисел: (10b + a) − (10a + b) = 9(b − a).
  • «Вторая цифра на 2 больше первой» → b = a + 2
  • «Вторая цифра на 3 меньше первой» → b = a − 3
  • «Сумма квадратов чисел равна 1130» → n² + m² = 1130
  • «Квадрат нового числа на 3960 больше» → m² − n² = 3960

Интерактивный разбор признаков делимости на 7, 11, 13, 17, 19

Выберите делитель и число. Система пошагово покажет применение признака делимости.
7: rest − 2·last
11: разность сумм по местам
13: rest + 4·last
17: rest − 5·last
19: rest + 2·last

Задача 1. Сумма квадратов после перестановки цифр

Задумали двузначное число. При перестановке цифр сумма квадратов полученного числа и исходного числа равна 1130. Вторая цифра на 2 больше первой. Найти исходное число.
1
Обозначим первую цифру через a. Тогда вторая цифра равна a + 2.
n = 10a + (a + 2) = 11a + 2
2
После перестановки цифр:
m = 10(a + 2) + a = 11a + 20
3
Составим уравнение:
(11a + 2)² + (11a + 20)² = 1130
4
После раскрытия скобок:
242a² + 484a − 726 = 0
a² + 2a − 3 = 0
5
Разложим на множители:
(a − 1)(a + 3) = 0
Подходит только a = 1. Тогда b = 3.
6
Проверка:
13² + 31² = 169 + 961 = 1130
Ответ: 13

Задача 2. Разность квадратов

Задумали двузначное число. Вторая цифра на 4 больше первой. После перестановки цифр квадрат нового числа оказался на 3960 больше квадрата исходного. Найти исходное число.
1
Пусть первая цифра равна a, тогда вторая цифра равна a + 4.
n = 11a + 4, m = 11a + 40
2
По условию:
m² − n² = 3960
(m − n)(m + n) = 3960
3
Находим множители:
m − n = 36
m + n = 22a + 44
4
Получаем:
36(22a + 44) = 3960
22a + 44 = 110
a = 3
5
Тогда b = 7, исходное число 37.
73² − 37² = 36 · 110 = 3960
Ответ: 37

Сводка

Что помнить Формула или правило
Запись двузначного числа 10a + b
Перестановка цифр 10b + a
«На k больше» b = a + k
«На k меньше» b = a − k
Разность чисел (10b + a) − (10a + b) = 9(b − a)
Разность квадратов x² − y² = (x − y)(x + y)
Делимость на 3 Сумма цифр делится на 3.
Делимость на 9 Сумма цифр делится на 9.
Делимость на 11 Разность суммы цифр на нечётных и чётных местах делится на 11.
Главная ошибка Двузначное число нельзя заменять суммой его цифр.
