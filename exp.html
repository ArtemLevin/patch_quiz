📖 Теория & методы

Неопределённый интеграл ∫ f(x) dx — семейство первообразных. Если знаменатель не зависит от x (y = const), то выносим константу:

∫ x/(y²–9) dx = 1/(y²–9) · ∫ x dx = x²/(2(y²–9)) + C

Если y = x (зависимость), получаем интеграл вида:

∫ x/(x²–9) dx = ½·∫ 2x/(x²–9) dx = ½·ln|x²–9| + C

✔️ Логарифмическое интегрирование: ∫ f'(x)/f(x) dx = ln|f(x)| + C

📌 Важно: при y = ±3 или x = ±3 интеграл расходится (особые точки).

✍️ Разбор задач

Задача 1 (параметр): Вычислить ∫ x/(25–9) dx при y=5.
Решение: y²–9 = 25–9 = 16 → ∫ x/16 dx = x²/32 + C
✅ Ответ: x²/32 + C
Задача 2 (y = x): Найти ∫ x/(x²–9) dx.
Решение: замена u = x²–9, du=2x dx → (1/2)∫ du/u = ½ ln|x²–9| + C
✅ Ответ: ½ ln|x²–9| + C
