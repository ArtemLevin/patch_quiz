Тема 1. Разложение на множители

В уравнениях вида (x − 2)(x² + 8x + 16) − 7(x + 4) = 0 сначала нужно упростить квадратный трёхчлен, затем вынести общий множитель, а уже после применять правило нуля произведения.

x² + 8x + 16 = (x + 4)² → (x + 4)((x − 2)(x + 4) − 7) = 0

Тема 2. Движение по таблице

В задачах на движение лучше выбрать переменную так, чтобы ответ читался прямо из условия. Если требуется скорость медленного, обозначаем именно её через x, а время выражаем формулой t = s / v.

120 / x − 120 / (x + 5) = 2

Тема 3. График с модулем и параметр

В выражениях, где встречаются и x², и |x|, удобно использовать равенство x² = |x|². Это помогает вынести |x| и сократить дробь. Но исходная ОДЗ при этом обязана сохраниться.

(|x| − 1,5x²) ≠ 0 → |x|(1 − 1,5|x|) ≠ 0

Техника мышления

  • Сначала ищем ФСУ и общий множитель.
  • Потом приводим выражение к простому виду.
  • После сокращения обязательно возвращаемся к ОДЗ.
  • В задачах с параметром исследуем геометрию пересечений, а не только алгебру.

Интерактивная визуализация графика

Рассматриваем функцию y = (1,5|x| − 1) / (|x| − 1,5x²). После упрощения она принимает вид y = −1 / |x|, но точки x = 0, x = ±2/3 остаются запрещёнными. Ниже можно двигать наклон прямой y = kx и видеть, когда общих точек нет.

график y = −1 / |x|
прямая y = kx
выколотые точки (±2/3; −3/2)
найденные пересечения
Наблюдение. При k = 0 прямая совпадает с осью Ox и не пересекает график, потому что весь график расположен ниже оси Ox.

Теория в коротких карточках

Ниже — концентрат правил, который удобно повторять перед практикой. Каждая карточка раскрывается и показывает только действительно нужное.

Алгоритм упрощения уравнения 5 быстрых проверок
  1. Проверить формулы сокращённого умножения.
  2. Проверить общий множитель.
  3. Разложить квадратный трёхчлен, если это возможно.
  4. Учитывать ОДЗ, если есть знаменатель или модуль.
  5. Стремиться к сокращению и более простому виду.
Главная идея: всё делается ради упрощения.
Квадратный трёхчлен и дискриминант D = b² − 4ac
Для x² + 8x + 16:
D = 8² − 4·1·16 = 64 − 64 = 0
Значит, корень один по значению, но кратность равна двум:
x₁ = x₂ = −8 / 2 = −4
Следовательно,
x² + 8x + 16 = (x + 4)²
Табличный метод в задачах на движение s = vt, t = s / v
Если нужно найти скорость медленного, удобно обозначить её через x. Тогда скорость быстрого станет x + 5, а времена будут равны:
t₁ = 120 / (x + 5), t₂ = 120 / x
По условию второй приехал на 2 часа позже:
120 / x − 120 / (x + 5) = 2
Модуль и ОДЗ x² = |x|²
В задаче с графиком знаменатель равен |x| − 1,5x². Поскольку x² = |x|², получаем:
|x| − 1,5|x|² = |x|(1 − 1,5|x|)
Поэтому запрещены значения:
x = 0, x = 2/3, x = −2/3
После сокращения график упрощается, но эти точки остаются выколотыми.

Ловушки, в которые легко попасть

Ловушка 1. Забыть множитель после D = 0 Если дискриминант равен нулю, трёхчлен не исчезает и не остаётся «без разложения». Он превращается в квадрат линейного множителя.
Ловушка 2. Потерять ОДЗ после сокращения В функции y = (1,5|x| − 1) / (|x| − 1,5x²) после сокращения получается y = −1 / |x|, но точки x = 0 и x = ±2/3 всё равно запрещены.
Ловушка 3. Неправильно делить дробь на число Если делим 120 / x на 2, то получаем не 120 / (x : 2), а:
120 / x : 2 = (120 / x) · 1/2 = 60 / x
Ловушка 4. Анализировать только алгебру, но не геометрию В параметрической задаче прямая может «касаться» графика только в выколотой точке. Формально такая точка не принадлежит графику, значит пересечения нет.

Подробные разборы

Ниже — два опорных решения: одно алгебраическое, второе графо-аналитическое. Их структура повторяет логику, которую стоит воспроизводить на экзамене.

Задача 1. Решить уравнение

(x − 2)(x² + 8x + 16) − 7(x + 4) = 0
Шаг 1. Разложим квадратный трёхчлен x² + 8x + 16 = (x + 4)², потому что D = 64 − 64 = 0 и корень x = −4.
Шаг 2. Подставим
(x − 2)(x + 4)² − 7(x + 4) = 0
Шаг 3. Вынесем общий множитель
(x + 4) · ((x − 2)(x + 4) − 7) = 0
Шаг 4. Упростим вторую скобку (x − 2)(x + 4) = x² + 2x − 8, значит:
(x + 4)(x² + 2x − 15) = 0
Шаг 5. Разложим ещё раз
x² + 2x − 15 = (x + 5)(x − 3)
Шаг 6. Применим правило нуля произведения
(x + 4)(x + 5)(x − 3) = 0
Ответ: x = −5, −4, 3

Задача 2. Для каких k прямая y = kx не имеет общих точек с графиком?

y = (1,5|x| − 1) / (|x| − 1,5x²)
Шаг 1. Найдём ОДЗ
|x| − 1,5x² ≠ 0 → |x|(1 − 1,5|x|) ≠ 0
Значит, x ≠ 0 и x ≠ ±2/3.
Шаг 2. Упростим функцию
1,5|x| − 1 = −(1 − 1,5|x|)
поэтому
y = −1 / |x|
но с выколотыми точками (2/3; −3/2) и (−2/3; −3/2).
Шаг 3. Исследуем прямую y = kx При k = 0 получаем ось Ox, а весь график находится ниже неё, значит пересечений нет.
Шаг 4. Проверим выколотые точки Для точки (2/3; −3/2):
−3/2 = k · 2/3 → k = −9/4
Для точки (−2/3; −3/2):
−3/2 = k · (−2/3) → k = 9/4
Итог
Ответ: k = −9/4, 0, 9/4

ФСУ

s = vt, t = s / v

Модуль

Удобный переход, когда нужно вынести |x| и сократить выражение.

ОДЗ сохраняется всегда

Нельзя возвращать в график точки, которые были запрещены исходным знаменателем.

Как использовать квиз

Отвечайте без подсказок, как будто это короткий устный опрос перед решением. Хороший результат — не просто число баллов, а способность быстро объяснить, почему ответ именно такой.

Проверяется
  • распознавание ФСУ
  • дискриминант и корни
  • логика табличного метода
  • понимание ОДЗ
  • геометрия прямой y = kx

Генератор тренировочных задач

Генератор использует только безопасные шаблоны с заранее корректной математикой. Можно получать новые уравнения, задачи на движение и графические вопросы по образцу изученных.

Выберите тип тренировки

Генератор строит только корректные задачи: уравнения с разложением на множители, задачи на движение с положительным ответом и графические вопросы с заранее известными запрещёнными точками.

Как устроена генерация

Тип 1. Уравнение Используется шаблон (x − m)(x + p)² − q(x + p) = 0, где q подобран так, чтобы второе квадратное уравнение имело целые корни.
Тип 2. Движение Сначала выбирается положительная скорость x, затем строится расстояние так, чтобы уравнение имело именно этот ответ.
Тип 3. Графика Берётся функция вида y = −A / |x| с выколотыми точками x = ±h. Ответом становятся три значения: −A / h², 0 и A / h².

Краткая печатная сводка

Основные результаты: x² + 8x + 16 = (x + 4)²; решение уравнения (x − 2)(x² + 8x + 16) − 7(x + 4) = 0 даёт x = −5, −4, 3; в задаче на движение скорость второго велосипедиста равна 15 км/ч; функция y = (1,5|x| − 1) / (|x| − 1,5x²) упрощается до y = −1 / |x| при x ≠ 0, ±2/3; прямая y = kx не имеет общих точек с графиком при k = −9/4, 0, 9/4.

Математика становится яснее, когда её можно увидеть, проверить и перестроить своими руками.
