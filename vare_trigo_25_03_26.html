Пульт параметра

Для уравнения tg x = a общий вид решения: x = arctg(a) + πn, где n ∈ ℤ.

Теория

Карточки-аккордеоны по теме

Здесь собрана логика всего блока: распознавание однородных уравнений, формулы приведения, двойной угол и стандартные записи корней.

1. Когда уравнение можно считать однородным распознавание

Основные шаблоны:

a·sin x + b·cos x = 0
a·sin²x + b·sin x·cos x + c·cos²x = 0

В обоих случаях удобно переходить к tan x, но только после отдельной проверки случаев cos x = 0.

2. Формулы приведения, без которых тема не работает приведение углов
sin(π/2 − x) = cos x
cos(3π/2 + x) = sin x
cos(x − π/2) = sin x

Сначала приводим все углы к одному типу, а уже потом выбираем формулу двойного угла или группировку.

3. Формулы двойного угла и полезные выборы выбор формы
sin 2x = 2sin x·cos x
cos 2x = cos²x − sin²x = 2cos²x − 1 = 1 − 2sin²x

Выбор формы для cos 2x делается по ситуации. Если в уравнении уже есть sin x, часто удобно использовать 1 − 2sin²x. Если доминирует cos x, часто удобнее взять 2cos²x − 1.

4. Когда можно делить на cos x или cos²x без потери корней

Деление допустимо только после явной проверки, что возможные корни при cos x = 0 не теряются.

Пример:
2cos x(cos x + sin x) = 0
Сначала: cos x = 0
Потом: cos x + sin x = 0

Нельзя сразу сокращать cos x, потому что тогда исчезнет целое семейство решений.

5. Стандартные формулы корней обязательный минимум
tg α = a ⇒ α = arctg(a) + πn

sin α = a ⇒ α = arcsin(a) + 2πn или α = π − arcsin(a) + 2πn

cos α = a ⇒ α = arccos(a) + 2πn или α = −arccos(a) + 2πn

Для отрицательных значений удобнее либо использовать знак минус прямо в формуле, либо сразу переходить к известным табличным углам, если число стандартное.

6. Что делать с выражением sin⁴t − cos⁴t алгебра внутри тригонометрии
sin⁴t − cos⁴t = (sin²t − cos²t)(sin²t + cos²t) = sin²t − cos²t = −cos 2t

Это один из самых быстрых способов резко упростить уравнение и привести его к сумме или произведению стандартных функций.

Разбор задач

Два эталонных разбора

Здесь важна не только техника, но и порядок действий: приведение, выбор формулы, разложение на множители, запись всех семейств корней.

Задача 1. 2sin²(π/2 − x) + sin 2x = 0

Шаг 1. Приведение
sin(π/2 − x) = cos x, sin 2x = 2sin x·cos x
Шаг 2. Подстановка
2cos²x + 2sin x·cos x = 0
Шаг 3. Вынесение множителя
2cos x(cos x + sin x) = 0
Шаг 4. Первый случай
cos x = 0 ⇒ x = π/2 + πn
Шаг 5. Второй случай
cos x + sin x = 0 ⇒ 1 + tg x = 0 ⇒ x = −π/4 + πn
Ответ
x = π/2 + πn или x = −π/4 + πn, n ∈ ℤ

Задача 2. cos 2x − √2cos(3π/2 + x) − 1 = 0

Шаг 1. Формула приведения
cos(3π/2 + x) = sin x
Шаг 2. Удобная форма для cos 2x
cos 2x = 1 − 2sin²x
Шаг 3. Подстановка
1 − 2sin²x − √2sin x − 1 = 0
2sin²x + √2sin x = 0
Шаг 4. Разложение на множители
sin x(2sin x + √2) = 0
Шаг 5. Решения
sin x = 0 ⇒ x = πn
sin x = −√2/2 ⇒ x = −π/4 + 2πn или x = 5π/4 + 2πn
