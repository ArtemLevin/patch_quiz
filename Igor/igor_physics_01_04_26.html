Идея решения

  • Тело падает без начальной скорости, значит путь за время t равен s = g·t²/2.
  • Каждая часть проходитcя за одно и то же время τ.
  • На втором и третьем участках тело уже летит с ненулевой начальной скоростью.
  • Из этого получается красивое отношение частей: h₁ : h₂ : h₃ = 1 : 3 : 5.
  • Сумма частей 1 + 3 + 5 = 9 долей, значит одна доля равна 270 / 9 = 30 м.
Ключевая мысль: при равных промежутках времени пути не равны, потому что скорость всё время растёт.

1) Что дано

Высота: h = 270 м. Падение свободное, без начальной скорости. Нужно найти h₁, h₂, h₃, если t₁ = t₂ = t₃.

2) Что означает условие

Высоту делят не на равные расстояния, а на такие части, которые проходятся за одинаковые времена. Так как тело разгоняется, за одинаковые промежутки времени оно проходит всё большие расстояния.

3) Интерпретация

Обозначим время прохождения каждого участка через τ. Тогда:

h₁ — путь за первые τ секунд; h₂ — путь за следующие τ секунд; h₃ — путь за последние τ секунд.

На первом участке начальная скорость равна 0. На втором участке начальная скорость уже равна конечной скорости первого участка. На третьем — конечной скорости второго участка.

4) Ключевая идея

1Первый участок. Без начальной скорости:
h₁ = g·τ²/2
2Скорость к концу первого участка:
v₁ = g·τ
3Второй участок. Теперь начальная скорость уже v₁:
h₂ = v₁·τ + g·τ²/2 = g·τ² + g·τ²/2 = 3g·τ²/2 = 3h₁
4Скорость к концу второго участка:
v₂ = v₁ + g·τ = 2g·τ
5Третий участок. Начальная скорость равна v₂:
h₃ = v₂·τ + g·τ²/2 = 2g·τ² + g·τ²/2 = 5g·τ²/2 = 5h₁

5) Пошаговое доведение до ответа

Из предыдущего пункта:

h₁ : h₂ : h₃ = 1 : 3 : 5

Сумма всех частей равна всей высоте:

h₁ + h₂ + h₃ = 270
1k + 3k + 5k = 270 ⇒ 9k = 270 ⇒ k = 30

Тогда:

h₁ = 30 м,   h₂ = 90 м,   h₃ = 150 м

6) Проверка ответа

Почему значения подходят

  • 30 : 90 : 150 = 1 : 3 : 5.
  • 30 + 90 + 150 = 270.
  • Это именно то отношение путей, которое даёт свободное падение за равные времена.

Почему других значений нет

  • Отношение 1 : 3 : 5 выводится из формул движения однозначно.
  • Общая высота фиксирована: 270 м.
  • Значит, одна доля и все три части определяются единственным образом.

7) Краткий вывод

При свободном падении без начальной скорости пути за три равных промежутка времени относятся как 1 : 3 : 5. Поэтому для высоты 270 м получаем: h₁ = 30 м, h₂ = 90 м, h₃ = 150 м.

Типичные ошибки

  • Считать, что раз времена равны, то и пути равны.
  • Для второго и третьего участка забывать начальную скорость, накопленную раньше.
  • Использовать только формулу s = g·t²/2 для каждого участка без учёта того, что это формула пути от начала падения.
  • Не проверять сумму h₁ + h₂ + h₃.
  • Путать весь путь от начала падения с путём на отдельном участке.
