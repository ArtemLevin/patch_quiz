Тема 1

Метод интервалов для многочленов

Универсальный метод решения рациональных неравенств, основанный на анализе знаков множителей

📐 Алгоритм решения

Пошаговая инструкция:

  • Найти корни уравнения f(x) = 0
  • Отметить корни на числовой прямой
  • Определить знак на правом интервале
  • Чередовать знаки при переходе через корни нечётной кратности
  • Выбрать нужные интервалы в соответствии со знаком неравенства
  • Для ≥ и ≤ включить корни, для > и < — исключить
⚠️ Ловушка

При корнях чётной кратности знак не меняется при переходе через корень. При корнях нечётной кратности знак меняется.

✏️ Задача 1: Базовая

Условие: Решите неравенство:

(x − 1)(x − 2)(x − 3)(x − 4) ≥ 0

Шаг 1. Найдём корни уравнения:

(x − 1)(x − 2)(x − 3)(x − 4) = 0
x₁ = 1, x₂ = 2, x₃ = 3, x₄ = 4

Все корни имеют кратность 1 (нечётная).

Шаг 2. Числовая прямая:

+ − + − +

Шаг 3. Определяем знак на правом интервале:

При x > 4 все множители положительны, значит произведение положительно.

Шаг 4. Выбираем интервалы:

Так как неравенство ≥ 0, выбираем интервалы со знаком + и включаем корни.

Ответ
x ∈ (−∞; 1] ∪ [2; 3] ∪ [4; +∞)

Тема 2

Исследование функций с модулями

Методика работы с кусочно-заданными функциями, содержащими выражения под знаком модуля

📊 Определение модуля

|a| = { a, если a ≥ 0
     { −a, если a < 0

🔍 Алгоритм исследования

Пошаговая инструкция:

  • Найти точки, где выражения под модулем равны нулю
  • Разбить область определения на интервалы по этим точкам
  • На каждом интервале раскрыть модули
  • Исследовать полученную кусочно-заданную функцию
⚠️ Ловушка

Не забывайте проверять границы интервалов — функция может быть непрерывной в точках «сшивки», но производная там не существует.

✏️ Задача 2: Исследование функции

Условие: Исследуйте функцию:

y = |x + 2| + |x − 3| − 4

Шаг 1. Найдём критические точки:

x + 2 = 0 ⇒ x = −2
x − 3 = 0 ⇒ x = 3

Шаг 2. Разобьём на интервалы:

Интервал Выражение
x < −2 y = −(x + 2) − (x − 3) − 4 = −2x − 3
−2 ≤ x < 3 y = (x + 2) − (x − 3) − 4 = 1
x ≥ 3 y = (x + 2) + (x − 3) − 4 = 2x − 5

Шаг 3. Исследуем функцию:

  • При x < −2: y = −2x − 3 — убывающая линейная функция
  • При −2 ≤ x < 3: y = 1 — постоянная функция
  • При x ≥ 3: y = 2x − 5 — возрастающая линейная функция

График функции:

Шаг 4. Выводы:

  • Область определения: D(y) = ℝ
  • Область значений: E(y) = [1; +∞)
  • Функция убывает на (−∞; −2], постоянна на [−2; 3], возрастает на [3; +∞)
  • Минимум: ymin = 1 при x ∈ [−2; 3]
  • Функция непрерывна на всей области определения
Ответ
D(y) = ℝ, E(y) = [1; +∞), ymin = 1 при x ∈ [−2; 3]

Тема 3

Задачи с параметрами

Графо-аналитический метод решения квадратных уравнений с параметром и условиями на корни

📈 Условия расположения корней

Для f(x) = ax² + bx + c на интервале (m; n):

  • D > 0 (два различных корня)
  • a · f(m) > 0 (значение на левой границе того же знака, что a)
  • a · f(n) > 0 (значение на правой границе того же знака, что a)
  • m < x₀ < n, где x₀ = −b / (2a) (абсцисса вершины внутри интервала)
⚠️ Ловушка

Не забывайте проверять все четыре условия одновременно. Пропуск любого условия приводит к потере решений или появлению посторонних.

✏️ Задача 3: Параметр

Условие: При каких значениях a уравнение

x² − 2ax + a + 3 = 0

имеет два различных корня, и оба принадлежат интервалу (0; 4)?

Шаг 1. Обозначим функцию:

f(x) = x² − 2ax + a + 3

Парабола ветвями вверх (a = 1 > 0).

Шаг 2. Условие 1: D > 0

D = (−2a)² − 4 · 1 · (a + 3) = 4a² − 4a − 12
D > 0 ⇒ 4a² − 4a − 12 > 0
a² − a − 3 > 0
Корни: a₁,₂ = (1 ± √13) / 2
a ∈ (−∞; (1 − √13) / 2) ∪ ((1 + √13) / 2; +∞)

Шаг 3. Условие 2: f(0) > 0

f(0) = 0² − 2a · 0 + a + 3 = a + 3
a + 3 > 0 ⇒ a > −3

Шаг 4. Условие 3: f(4) > 0

f(4) = 16 − 8a + a + 3 = 19 − 7a
19 − 7a > 0 ⇒ a < 19 / 7

Шаг 5. Условие 4: Вершина внутри (0; 4)

x₀ = 2a / 2 = a
0 < a < 4

Шаг 6. Объединяем все условия:

Условие Решение
D > 0 a ∈ (−∞; (1 − √13) / 2) ∪ ((1 + √13) / 2; +∞)
f(0) > 0 a > −3
f(4) > 0 a < 19 / 7
0 < x₀ < 4 0 < a < 4
Пересечение a ∈ ((1 + √13) / 2; 19 / 7)

Шаг 7. Проверка границ:

(1 + √13) / 2 ≈ 2.3, 19 / 7 ≈ 2.71

Интервал непустой.

Ответ
a ∈ ((1 + √13) / 2; 19 / 7)
