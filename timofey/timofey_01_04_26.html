1Определяем, в какую фигуру входит искомая величина (угол, отрезок, площадь)

2Записываем все известные свойства фигуры (сумма углов, равенство сторон, параллельность)

3Ищем прямоугольные треугольники для применения тригонометрии

1Что Дано

Рассмотрим классическую задачу на равнобедренный треугольник с биссектрисой:

ΔABC: AB = BC (равнобедренный)
∠ABC = 76°
AF — биссектриса ∠BAC
Найти: ∠AFC = ?
💡 Нажмите, чтобы увидеть ключевые свойства равнобедренного треугольника

✓ Углы при основании равны: ∠BAC = ∠BCA

✓ Сумма углов треугольника: ∠A + ∠B + ∠C = 180°

✓ Биссектриса делит угол пополам

2Анализ Условий

Из условия AB = BC следует:

∠BAC = ∠BCA = (180° - 76°) / 2 = 52°

Так как AF — биссектриса:

∠BAF = ∠FAC = 52° / 2 = 26°
🔍 Подсветить углы на чертеже

3Ключевая Идея

Рассматриваем треугольник ΔAFC, в который входит искомый угол:

∠AFC + ∠FAC + ∠FCA = 180°

Откуда:

∠AFC = 180° - ∠FAC - ∠FCA
🎯 Выделить треугольник AFC на чертеже

4Вычисления

Подставляем известные значения:

∠FAC = 26° (половина ∠BAC)
∠FCA = 52° (равен ∠BCA)
∠AFC = 180° - 26° - 52° = 102°

5Проверка Ответа

✓ Сумма углов ΔAFC: 102° + 26° + 52° = 180°

✓ Угол тупой, что соответствует чертежу

✓ Все условия задачи выполнены

Ответ: 102°

⚠️ Типичные Ошибки

! Ошибка 1: Неправильное применение свойства биссектрисы

Некоторые забывают, что биссектриса делит угол пополам, и используют полное значение угла.

! Ошибка 2: Путаница с углами при основании

В равнобедренном треугольнике равны углы при основании, а не при вершине.

! Ошибка 3: Арифметические ошибки

При вычислении (180 - 76) / 2 часто ошибаются. Проверьте: 104 / 2 = 52°
