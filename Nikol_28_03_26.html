Теория в карточках

Открой нужный блок. Каждая карточка отвечает за один тип ошибки, который на экзамене особенно опасен.

1. Уравнения с корнем: что такое фильтрующее условие

Для уравнения вида √(F(x)) = G(x) мало проверить только подкоренное выражение.

Нужно записать два условия:

F(x) ≥ 0 и G(x) ≥ 0

Второе условие называется фильтрующим, потому что после возведения в квадрат оно отсекает посторонние корни.

Алгоритм:

  1. записать ОДЗ для корня;
  2. записать неотрицательность правой части;
  3. возвести в квадрат;
  4. решить полученное уравнение;
  5. отобрать корни по ограничениям.
2. Логарифм и тригонометрия: где решения не могут существовать

Для логарифма logₐ b обязательно нужно:

a > 0, a ≠ 1, b > 0

Если основание — обычное число, например 2, то проверяем прежде всего аргумент логарифма.

Для уравнения log₂(cos x) = c требуется:

cos x > 0

После перехода к cos x = 2c обязательно сравниваем правую часть с диапазоном косинуса:

−1 ≤ cos x ≤ 1

Если справа получилось число больше 1, решений нет сразу.

3. Графики прямых: как быстро восстановить формулу

Если прямая проходит через две точки, удобно использовать вид:

y = kx + b

Подставляем координаты двух точек, получаем систему на k и b.

Дальше для точки пересечения двух прямых приравниваем их правые части.

Важно не путать:

  • абсцисса — это x;
  • ордината — это y.
4. Задачи на производительность: только через таблицу

Весь объём работы принимаем за 1. Тогда:

производительность = результат / время

Если два объекта работают вместе, то:

совместная производительность = сумма отдельных производительностей

Для бассейна, трубы, заказа, принтера или насоса схема одна и та же. Наиболее частая ошибка — неверно перевести минуты в часы.

7 ч 22 мин = 7 + 22/60 = 7 + 11/30 = 221/30 ч
5. Самые важные диапазоны и быстрые проверки

Синус и косинус

−1 ≤ sin x ≤ 1 −1 ≤ cos x ≤ 1

Квадраты

0 ≤ sin²x ≤ 1 0 ≤ cos²x ≤ 1

Тангенс

tan x — любое действительное число там, где определён

Разборы задач

Два полных примера из пособия: один про корень и отбор корней, второй — про логарифм с тригонометрией.

Задача 1. Решить уравнение √(1 − x) = x + 2

1
Запишем ограничения:
1 − x ≥ 0 ⇒ x ≤ 1
x + 2 ≥ 0 ⇒ x ≥ −2
Значит, искать решения имеет смысл только на отрезке −2 ≤ x ≤ 1
2
Возводим обе части в квадрат:
1 − x = (x + 2)²
1 − x = x² + 4x + 4
x² + 5x + 3 = 0
3
Решаем квадратное уравнение:
D = 25 − 12 = 13
x = (−5 ± √13) / 2
4
Проверяем ограничения:
(−5 − √13) / 2 < −2 — не подходит
(−5 + √13) / 2 ∈ [−2; 1] — подходит
Ответ: x = (−5 + √13) / 2

Задача 2. Решить уравнение log₂(cos x) = −½

1
Сначала записываем ОДЗ логарифма:
cos x > 0
2
Переходим к показательному виду:
cos x = 2−1/2 = 1 / √2 = √2 / 2
3
Это табличное значение косинуса. На единичной окружности получаем:
x = π/4 + 2πn или x = −π/4 + 2πn, n ∈ ℤ
4
ОДЗ выполнено автоматически, потому что √2 / 2 > 0
Ответ: x = π/4 + 2πn или x = −π/4 + 2πn, n ∈ ℤ

Короткий разбор текстовой задачи

Две трубы вместе наполняют бассейн за 7 ч 22 мин, первая — за 13 ч. Найти время второй трубы.

Объект Время Производительность Результат
Первая труба 13 ч 1/13 1 бассейн
Вторая труба x ч 1/x 1 бассейн
Обе вместе 221/30 ч 30/221 1 бассейн
1/13 + 1/x = 30/221 ⇒ 1/x = 13/221 ⇒ x = 17
Ответ: 17 часов

Корни и отбор решений

  • Для √(F(x)) нужно F(x) ≥ 0.
  • Для уравнения √(F(x)) = G(x) добавляй G(x) ≥ 0.
  • После возведения в квадрат всегда делай отбор корней.
√(1 − x) = x + 2 ⇒ x ≤ 1 и x ≥ −2

Логарифмы

  • logₐ b существует только при a > 0, a ≠ 1, b > 0.
  • Если есть log₂(cos x), то обязательно cos x > 0.
  • Дальше сравни результат с диапазоном cos x.
−1 ≤ cos x ≤ 1

Графики прямых

  • Используй форму y = kx + b.
  • Две точки дают систему на k и b.
  • Точка пересечения — это равенство правых частей.
3x − 4 = −2x + 3 ⇒ x = 7/5, y = 1/5

Производительность

  • Весь объём работы принимается за 1.
  • Производительность = 1 / время.
  • Вместе: 1/t₁ + 1/t₂ = 1/tсовм.
1/13 + 1/x = 30/221 ⇒ x = 17
