Теория

Ключевые свойства без шумов и лишней тяжести

Каждый блок раскрывается отдельно, как карточка памяти перед задачей.

Центр правильного шестиугольника делит фигуру на шесть равносторонних треугольников. Отсюда почти мгновенно следуют формулы для радиусов, диагоналей и площади.
Короткая диагональ соединяет вершины через одну и равна a√3. Длинная диагональ соединяет противоположные вершины и равна 2a, потому что это диаметр описанной окружности.
Площадь правильного шестиугольника равна сумме площадей шести равносторонних треугольников: S = (3√3 / 2)a². Треугольник по вершинам через одну тоже равносторонний и имеет площадь, равную половине площади шестиугольника.
  • Не путать R и r.
  • Не путать a√3 и 2a.
  • Расстояние между противоположными сторонами равно 2r, а не R.
  • Треугольник из вершин через одну даёт ровно половину площади шестиугольника.
Разбор

Задача 1

Сторона известна, значит почти всё извлекается прямым применением формул.

Условие. В правильном шестиугольнике сторона равна 6 см. Найдите диагонали, радиусы окружностей, углы, площадь шестиугольника и площадь треугольника ACE.

  1. Короткая диагональ: AC = a√3 = 6√3.
  2. Длинная диагональ: AD = 2a = 12.
  3. Радиусы: R = 6, r = 3√3.
  4. В треугольнике ABC угол при B равен 120°, значит ∠BAC = 30°.
  5. AD — диаметр описанной окружности, поэтому ∠ACD = 90°.
  6. В прямоугольном треугольнике ACD: sin ∠CAD = CD / AD = 6 / 12 = 1/2, значит ∠CAD = 30°.
  7. Площадь шестиугольника: S = (3√3 / 2)·6² = 54√3.
  8. Площадь треугольника ACE вдвое меньше: 27√3.
Ответ. AC = 6√3, AD = 12, R = 6, r = 3√3, ∠BAC = 30°, ∠CAD = 30°, ∠ACD = 90°, S = 54√3, S(ACE) = 27√3.
Разбор

Задача 2

Дано расстояние между противоположными сторонами, значит сначала восстанавливаем апофему.

Условие. Расстояние между противоположными сторонами равно 12√3 см. Найдите сторону, диагонали, периметр, площадь шестиугольника, площадь описанного круга и площадь части круга вне шестиугольника.

  1. Расстояние между противоположными сторонами равно 2r, значит r = 6√3.
  2. По формуле r = (√3 / 2)a получаем a = 12.
  3. Короткая диагональ: 12√3. Длинная диагональ: 24.
  4. Периметр: P = 72.
  5. Площадь: S = 216√3.
  6. Радиус описанной окружности: R = 12, площадь круга: 144π.
  7. Часть круга вне шестиугольника: 144π − 216√3.
Ответ. a = 12, dкор = 12√3, dдл = 24, P = 72, S = 216√3, Sкруга = 144π, Sвне = 144π − 216√3.
