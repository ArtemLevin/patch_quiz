Числовая прямая для вложенных событий

Измени вероятности левого и правого луча и наблюдай, как автоматически пересчитывается средний интервал. Это ключевой приём для задач вида «ниже», «выше», «между».

Левый луч
Искомый интервал
Правый луч

Пульт модели

В этой схеме три непересекающиеся части полностью покрывают все исходы, поэтому P(середина) = 1 − P(левый луч) − P(правый луч).

Левый луч 0,27
Правый луч 0,36
P(18 ≤ X ≤ 21) = 1 − 0,27 − 0,36 = 0,37
Когда это работает
Если левая часть, средняя часть и правая часть не пересекаются и вместе дают всё пространство исходов.
Когда можно вычитать
Если одно событие вложено в другое. Например, P(9 ≤ X < 18) = P(X < 18) − P(X < 9).
Сигнал ошибки
Если сумма левого и правого луча больше 1, условие противоречиво: среднего интервала уже не остаётся.
Калькулятор

Дерево вероятностей для двух попыток

Здесь можно моделировать задачу про выстрелы или серии попыток. Сначала вычисляется вероятность успеха для одного объекта, затем — вероятность ровно одного успеха из трёх одинаковых объектов.

Настройка вероятностей

p₁0,50
p₂0,60
p = 0,50 + (1 − 0,50) · 0,60 = 0,80
q = 1 − 0,80 = 0,20
P(ровно одна из трёх) = 3 · 0,80 · 0,20² = 0,096
Формула успеха: p = p₁ + (1 − p₁)·p₂ Формула ровно одной удачи: 3·p·q²

Смысл формулы

Успех для одной мишени
p = 0,50 + 0,50 · 0,60 = 0,80
Неуспех для одной мишени
q = 1 − 0,80 = 0,20
Три взаимно исключающие схемы
УНН, НУН, ННУ → каждая имеет вероятность p·q², всего 3 таких случая.
Итог
P = 3 · 0,80 · 0,20² = 0,096
Теория

Карточки-аккордеоны с ключевыми идеями

Это сжатая, но полная карта темы. Открывай по мере необходимости: сначала модель, затем формула, затем смысл.

1. Как узнать, какую модель использовать основа темы

Первый вопрос в любой задаче: что здесь моделируется? Не начинай с вычислений.

  • Числовая прямая: если в условии есть «ниже», «выше», «между», «от ... до ...».
  • Дерево вероятностей: если процесс идёт по шагам и после результата одного шага возможен следующий.
  • Классическая вероятность: если исходы равновозможны, например места в группе, позиции, порядок.
  • Смешивание потоков: если объекты приходят из разных источников с разными вероятностями брака.
2. Вложенные события и дополнение до 1 лучи и интервалы

Если пространство исходов разбито на несколько непересекающихся частей, их вероятности складываются в 1.

P(середина) = 1 − P(левый луч) − P(правый луч)

Если одно событие полностью содержится в другом, можно вычитать:

Если A ⊂ B, то P(B \ A) = P(B) − P(A)
  • Пример: P(9 ≤ X < 18) = P(X < 18) − P(X < 9).
  • Пример: P(ровно 10 задач) = P(больше 9 задач) − P(больше 10 задач).
3. Дерево вероятностей: умножение и сложение пошаговый процесс

По одной ветви дерева вероятности умножаются, а вероятности альтернативных путей одного события складываются.

p = p₁ + (1 − p₁)·p₂
P(ровно одна удача из трёх) = 3·p·(1 − p)²
  • Сначала вычисляй вероятность успеха для одного объекта.
  • Потом переходи к трём объектам.
  • Не путай «ровно один успех» и «хотя бы один успех».
4. Как достать скрытую вероятность из процентов лотерея и доли

Если выигрышных билетов на 20% меньше, чем невыигрышных, удобно взять число невыигрышных за x.

Невыигрышных = x
Выигрышных = 0,8x
Всего = 1,8x
P(выигрыш) = 0,8x / 1,8x = 4/9

Для серии покупок безопаснее использовать дополнение:

P(хотя бы один выигрыш из n билетов) = 1 − (5/9)ⁿ
5. Классическая вероятность и равновозможные места группы и порядок

Когда каждый вариант расположения одинаково возможен, применяй схему:

P(A) = число благоприятных исходов / число всех исходов
  • При разбивке на две равные группы по 13 из 26 человек: после фиксации первого друга у второго 25 мест, из них 13 в другой группе и 12 в той же.
  • Вероятность оказаться в разных группах: 13/25.
  • Если первым выступает спортсмен из 50, а из Мексики 11 спортсменов, то вероятность равна 11/50.
6. Потоки из разных источников взвешенное среднее

Если первая линия выпускает в 3 раза больше продукции, чем вторая, то доли на складе равны 3/4 и 1/4.

P(брак) = (3/4)·0,10 + (1/4)·0,06 = 0,09
P(не брак) = 1 − 0,09 = 0,91

Это не обычное среднее, а взвешенное: источник с большей долей сильнее влияет на итог.

Разбор задач

Два эталонных решения из темы

Здесь важна не только формула, но и логика: почему именно эта модель выбрана и почему операция законна.

Задача 1. Температура между 18° и 21°

Известно: P(X < 18) = 0,27, P(X > 21) = 0,36. Найти P(18 ≤ X ≤ 21).

Шаг 1. Модель Это задача на числовую прямую: левый луч, средний интервал, правый луч.
Шаг 2. Разбиение всего пространства Эти три части не пересекаются и вместе дают все исходы.
Шаг 3. Формула
P(18 ≤ X ≤ 21) = 1 − 0,27 − 0,36
Ответ
P(18 ≤ X ≤ 21) = 0,37

Задача 2. Ровно одна поражённая мишень из трёх

Вероятность попадания первым выстрелом 0,5. Если первый выстрел неудачен, возможен второй, с вероятностью попадания 0,6.

Шаг 1. Найдём успех для одной мишени
p = 0,5 + 0,5 · 0,6 = 0,8
Шаг 2. Найдём неуспех для одной мишени
q = 1 − 0,8 = 0,2
Шаг 3. Ровно одна удача из трёх Возможны схемы УНН, НУН, ННУ.
P = 3 · 0,8 · 0,2²
Ответ
P = 0,096

Быстрая шпаргалка по теме

Это компактная таблица решений: какой тип задачи, какая модель, какая формула, где чаще всего ошибаются.

Тип задачи Главная модель Ключевая формула Типичная ошибка
«Между», «от ... до ...» Числовая прямая P(середина) = 1 − P(левый луч) − P(правый луч) Забывают, что части должны быть непересекающимися и покрывать всё пространство.
Вложенные события Разность вероятностей Если A ⊂ B, то P(B \ A) = P(B) − P(A) Вычитают вероятности, не проверив вложенность.
Два выстрела / две попытки Дерево вероятностей p = p₁ + (1 − p₁)·p₂ Складывают вероятности на одном пути вместо умножения.
Ровно одна удача из трёх Сумма трёх исключающих схем 3·p·q² Путают с «хотя бы одна удача».
Лотерея, серии покупок Дополнение P(хотя бы один) = 1 − qⁿ Слишком рано начинают суммировать случаи вручную.
Группы и места Классическая вероятность P(A) = благоприятные / все Не фиксируют одного участника и считают лишние варианты.
Две линии, два источника Взвешенное среднее P = α·p₁ + β·p₂ Берут простое среднее вместо учёта долей.
