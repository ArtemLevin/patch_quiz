Ровно три точки пересечения

Функция задачи

y = 2 − |x|, если x ≤ 4
y = −x² + 10x − 25, если x > 4

Удобный вид

y = 2 − |x|
y = −(x − 5)²

Вторая ветвь — это парабола с вершиной в точке (5; 0), но она берётся только при x > 4. Поэтому точка (4; −1), которая лежала бы на полной параболе, в график не входит.

Что нужно найти

Для каких m прямая y = m
имеет ровно 3 общие точки с графиком?

Почему эта задача красивая

Она учит смотреть на параметр не как на букву, а как на движущуюся горизонтальную прямую. Вместо тяжёлых алгебраических преобразований мы наблюдаем геометрию уравнения f(x) = m.

Главная мысль

Каждое решение уравнения f(x) = m — это абсцисса точки пересечения графика функции с горизонтальной прямой y = m. Значит, число решений уравнения равно числу пересечений.

видим, а не угадываем
учитываем ограничения x ≤ 4 и x > 4
следим за выколотой точкой

Что особенно важно

Первая ветвь

y = 2 − |x| — это ломаная с вершиной (0; 2). При x = 4 она даёт точку (4; −2), и эта точка входит в график.

Вторая ветвь

y = −(x − 5)² — это парабола, но только её часть при x > 4. Поэтому при уровне m = −1 остаётся только одна точка x = 6, а x = 4 не подходит.

Переходы по уровням

Количество точек меняется только на особых высотах: m = 2, m = 0, m = −1, m = −2. Их нельзя пропускать.

Теория без перегруза

Здесь собраны ровно те идеи, которые нужны для уверенного решения задачи.

1. Почему уравнение f(x) = m удобно решать графически
Горизонтальная прямая y = m пересекает график функции в тех точках, где ордината графика равна m. Значит, число решений уравнения f(x) = m равно числу общих точек графика и прямой y = m.
2. Как читать график y = 2 − |x|
Это перевёрнутая «галочка» с вершиной в точке (0; 2). Слева и справа график идёт вниз. Но в задаче берётся только часть при x ≤ 4, поэтому вся левая ветвь входит полностью, а справа график обрезается в точке (4; −2).
3. Как читать график y = −x² + 10x − 25
После выделения квадрата получаем y = −(x − 5)². Это парабола с вершиной (5; 0), ветви направлены вниз. Но по условию берётся только область x > 4. Поэтому точка (4; −1) не входит, хотя лежала бы на полной параболе.
4. Какие уровни m самые важные
Уровни m = 2, 0, −1, −2 — критические. На них либо прямая касается одной из ветвей, либо проходит через граничные точки, либо меняется количество решений. Проверка этих уровней обязательна.

Ловушки

Нельзя забывать про x > 4

Для параболы уровень m = −1 даёт две точки у полной параболы: x = 4 и x = 6. Но точка x = 4 запрещена условием.

Нельзя игнорировать точку x = 4 у первой ветви

Для y = 2 − |x| при x ≤ 4 точка (4; −2) входит в график. Значит, при m = −2 первая ветвь даёт две точки.

Нельзя делать вывод «почти так же»

В задачах с параметром интервалы и отдельные точки нужно записывать аккуратно. Здесь ответ не просто интервал, а объединение интервала и точки: [−2; −1] ∪ {0}.

Пошаговый разбор

Два режима: полный ход решения и отдельный разбор по ветвям.

Перепишем вторую формулу в удобном виде

y = −x² + 10x − 25 = −(x − 5)². Теперь ясно, что это парабола с вершиной (5; 0), и сразу видно, что её наибольшее значение равно 0.

Опишем первую ветвь

y = 2 − |x| при x ≤ 4. Это ломаная с вершиной (0; 2). При x = 4 получаем y = −2, значит точка (4; −2) принадлежит графику.

Опишем вторую ветвь

y = −(x − 5)² при x > 4. При x = 4 формула дала бы y = −1, но точка (4; −1) не входит в график, потому что условие строгое: x > 4.

Считаем решения на первой ветви

Уравнение 2 − |x| = m равносильно |x| = 2 − m.
Отсюда: m > 2 → 0 точек;
m = 2 → 1 точка;
−2 < m < 2 → 2 точки;
m = −2 → 2 точки;
m < −2 → 1 точка.

Считаем решения на второй ветви

Уравнение −(x − 5)² = m равносильно (x − 5)² = −m.
Отсюда: m > 0 → 0 точек;
m = 0 → 1 точка;
−1 < m < 0 → 2 точки;
m = −1 → 1 точка;
m < −1 → 1 точка.

Собираем общий ответ

Ровно 3 точки получаются в трёх случаях: при m = 0, при m = −1 и для всех m из интервала (−2; −1). Кроме того, при m = −2 тоже ровно 3 точки. Значит, m ∈ [−2; −1] ∪ {0}.

Ветвь 1: y = 2 − |x|, x ≤ 4

2 − |x| = m
|x| = 2 − m

Чтобы были решения, нужно 2 − m ≥ 0, то есть m ≤ 2. Обычно уравнение |x| = a даёт две точки x = ±a, но здесь есть ограничение x ≤ 4. Поэтому для очень низких уровней правая точка может выйти за границу.

m > 2 → 0
m = 2 → 1
−2 < m < 2 → 2
m = −2 → 2
m < −2 → 1

Ветвь 2: y = −(x − 5)², x > 4

−(x − 5)² = m
(x − 5)² = −m

Здесь обязательно m ≤ 0. У полной параболы обычно две симметричные точки, но из-за условия x > 4 левая точка может исчезнуть. Особенно важен уровень m = −1: он даёт x = 4 и x = 6, но x = 4 нельзя брать.

m > 0 → 0
m = 0 → 1
−1 < m < 0 → 2
m = −1 → 1
m < −1 → 1

Итоговая таблица

m > 2 → 0 + 0 = 0
m = 2 → 1 + 0 = 1
0 < m < 2 → 2 + 0 = 2
m = 0 → 2 + 1 = 3
−1 < m < 0 → 2 + 2 = 4
m = −1 → 2 + 1 = 3
−2 < m < −1 → 2 + 1 = 3
m = −2 → 2 + 1 = 3
m < −2 → 1 + 1 = 2

Финальный ответ

Прямая y = m имеет ровно три общие точки с графиком тогда и только тогда, когда
m ∈ [−2; −1] ∪ {0}
Это объединение замкнутого интервала и одной отдельной точки. Такая запись обязательна.

Сводка: что держать в голове

Компактный набор правил, который можно использовать как шпаргалку перед контрольной.

1. Параметр как высота

Уравнение f(x) = m означает: найти все точки графика, лежащие на высоте m. Поэтому анализируй не букву, а положение прямой y = m.

2. Абсолютная величина

  • |x| = a при a > 0 даёт 2 решения.
  • |x| = 0 даёт 1 решение.
  • |x| = a при a < 0 решений не имеет.

3. Парабола вида −(x − c)²

  • максимум равен 0;
  • для уровней ниже вершины обычно 2 точки;
  • но ограничение по x может убрать одну из них.

4. Границы важнее всего

Обязательно проверяй уровни, связанные с вершинами и граничными точками: здесь это m = 2, 0, −1, −2.

Генератор похожих задач

Генератор использует только безопасные шаблоны той же структуры, поэтому условия и ответы всегда корректны.

Как устроен шаблон

Генерируется функция вида:

y = A − |x|, при x ≤ B
y = −(x − C)², при x > B

Причём всегда берётся C = B + 1. Тогда в точке x = B вторая формула давала бы уровень −1, но эта точка не входит. Из-за этого ответ сохраняет красивую структуру:

m ∈ [A − B; −1] ∪ {0}

Это прямое и корректное обобщение основной задачи.
